# CHAPTER07 - 의사결정트리

- 주제: 의사결정트리
- 목표: 주가 방향 예측에서 규칙 기반 분기와 특성 중요도를 해석합니다.
- 실행: `python practice.py`

---

## 핵심 개념

- **분기 규칙**: 질문을 반복해서 데이터를 나누는 구조입니다.
- **해석 가능성**: 어떤 조건이 먼저 사용됐는지 사람이 읽기 쉽습니다.
- **과적합 주의**: 가지가 너무 깊어지면 학습 데이터만 잘 맞는 모델이 됩니다.

## 언제 쓰나요?

- 규칙 형태로 설명 가능한 모델이 필요할 때
- 숫자/범주형 특성이 섞여 있어도 빠르게 기준을 만들고 싶을 때
- 앙상블 모델(Random Forest, GBM)로 확장하기 전에 기초 구조를 볼 때

## 결과 해석 체크리스트

- 실행 결과의 `feature_importance_sum`이 1에 가까운지 확인합니다.
- 트리가 깊어질수록 설명은 쉬워도 일반화 성능이 떨어질 수 있음을 기억합니다.
- 로지스틱 회귀와 달리 비선형 규칙을 표현할 수 있다는 점을 비교합니다.

## 웹앱 실습 순서

1. 설명 탭에서 질문-분기 구조를 이해합니다.
2. 실행 후 결과 탭에서 중요도 관련 값을 확인합니다.
3. `chapter08` 랜덤포레스트와 연속해서 실행해 단일 트리와 앙상블 차이를 비교합니다.
4. 필요하면 예측 실험실에서 트리 기반 모델 결과와 함께 해석합니다.

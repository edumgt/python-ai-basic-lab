# CHAPTER109 - 주식 클러스터링과 군집 해석

- 주제: 주식 특징 기반 군집화
- 10분 기본 지식: 클러스터링은 비슷한 속성의 종목을 그룹으로 묶습니다.
- 30분 Python 실습: 수익률/변동성/가치지표로 K-Means 군집화와 그룹 해석을 수행합니다.
- 실행: `python practice.py`

---

## 핵심 개념

- **종목 군집화**: 수익률 패턴이 비슷한 종목끼리 묶어 시장 구조를 파악합니다.
- **그룹 해석**: 성장주, 방어주, 고변동주 같은 이름을 붙일 수 있습니다.
- **포트폴리오 보조 도구**: 서로 다른 군집을 섞어 분산 효과를 얻을 수 있습니다.

## 언제 쓰나요?

- 종목 수가 많아 비슷한 움직임을 자동으로 정리하고 싶을 때
- 대표 종목을 뽑거나 섹터 유사성을 확인하고 싶을 때
- 포트폴리오 구성 전에 시장 지형도를 보고 싶을 때

## 결과 해석 체크리스트

- 군집 수가 지나치게 많거나 적지 않은지 확인합니다.
- 같은 군집 안 종목들의 공통 특징을 설명해봅니다.
- 군집은 정답이 아니라 해석 도구이므로 시장 상황에 따라 달라질 수 있습니다.

## 웹앱 실습 순서

1. 실행 후 군집 결과와 그룹 해석을 먼저 확인합니다.
2. `chapter09` 기본 군집화와 차이를 비교합니다.
3. 포트폴리오 최적화(`chapter108`)와 함께 어떤 군집을 섞을지 생각합니다.
4. 필요하면 예측 실험실 결과를 보며 같은 섹터/종목 간 상관을 비교합니다.

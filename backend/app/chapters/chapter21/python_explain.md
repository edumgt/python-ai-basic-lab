## 먼저 기억해요
- `python 파일이름.py` 로 실행해요.
- 에러가 나면 메시지를 읽고 천천히 한 줄씩 확인해요.
- `run()` 함수가 있으면 그 함수가 핵심 실습이에요.

## 이 챕터에서 배우는 말

| 말 | 쉬운 뜻 |
|------|------|
| **스칼라** | 숫자 하나 |
| **벡터** | 숫자 여러 개를 묶은 것 |
| **선형변환** | 벡터에 가중치를 적용해 새 점수를 만드는 계산 |
| **가중합** | 입력마다 중요도를 곱해서 더한 값 |
| **편향** | 마지막에 더하는 보정값 |
| **시그모이드 함수** | 점수를 0~1 사이 값으로 바꾸는 함수 |
| **활성화 함수** | 뉴런 출력 모양을 결정하는 함수 |
| **오차** | 예측과 정답의 차이 |
| **비용(손실)** | 오차를 한 숫자로 요약한 값 |

이 문서는 그 값을 이용해  
**벡터 입력 → 선형변환 → 활성화 함수 → 오차 → 비용 → 수정** 흐름을 배우는 실습입니다.

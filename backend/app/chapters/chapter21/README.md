# CHAPTER21 - 신경망 기초와 학습(Forward/Backward)

- 주제: 신경망 모델 기초
- 목표: 주가 특성을 입력으로 가중치 행렬, 순전파, 손실 계산, 역전파, 경사하강법 업데이트를 한 흐름으로 실습합니다.
- 실행: `python practice.py`

---

## 핵심 개념

- **스칼라와 벡터**: 숫자 하나는 스칼라, 숫자 묶음은 벡터입니다. 신경망은 입력 벡터를 받아 계산합니다.
- **선형변환**: 입력 벡터에 가중치 행렬과 편향을 적용해 다음 층의 점수를 만드는 계산입니다.
- **가중합과 편향**: 각 뉴런은 입력값에 가중치를 곱해 더한 뒤, 편향을 더해 다음 계산의 기준점을 만듭니다.
- **시그모이드와 활성화 함수**: 선형변환 결과를 그대로 쓰지 않고, 시그모이드·ReLU 같은 활성화 함수로 바꿔 비선형 판단을 만듭니다.
- **오차와 비용**: 예측값과 실제값의 차이가 오차이고, 이 오차를 숫자 하나로 요약한 값이 손실/비용입니다.
- **순전파(Forward Pass)**: 입력 데이터가 입력층 → 은닉층 → 출력층 방향으로 흘러가며 예측값을 만드는 과정입니다.  
  예) 주가·거래량·이평선 입력 → 64개 뉴런이 패턴 계산 → 32개 뉴런이 판단 → 상승 확률 출력

- **역전파(Backpropagation)**: 예측이 틀렸을 때 오차를 **출력층 → 은닉층 → 입력층** 방향으로 거꾸로 전달하며 가중치를 수정하는 과정입니다.  
  예) 상승 예측이 틀리면, 출력 뉴런부터 입력 쪽까지 각 뉴런의 가중치를 "이 방향으로 틀렸으니 조금 수정"하며 개선합니다.

- **경사하강법(Gradient Descent)**: 역전파로 계산된 기울기를 이용해 손실이 줄어드는 방향으로 가중치를 조금씩 이동하는 학습 방법입니다.  
  순전파 → 오차 계산 → 역전파 → 가중치 수정의 **피드백 루프**를 수천 번 반복해 모델이 점점 정확해집니다.

## 언제 쓰나요?

- 비선형 패턴이 많아 선형 모델만으로 부족할 때
- 입력 특성이 많고 조합 규칙을 직접 만들기 어려울 때
- CNN, RNN, Transformer 같은 딥러닝 모델의 출발점을 이해할 때

## 결과 해석 체크리스트

- `입력 벡터 → 선형변환 → 활성화 함수` 흐름을 말로 설명할 수 있는지 확인합니다.
- 한 뉴런 기준으로 `가중합 → 편향 → 예측 → 오차 → 비용` 흐름을 말로 설명할 수 있는지 확인합니다.
- `initial_loss`와 `final_loss` 차이가 충분한지 확인합니다.
- `train_accuracy`가 높아졌다면 실제로 가중치가 학습된 것입니다.
- `weight_shapes`와 `softmax_example`을 보며 층 구조를 해석합니다.

## 웹앱 실습 순서

1. 설명 탭에서 가중합, 편향, 오차, 비용이 무엇인지 먼저 정리합니다.
2. 스칼라, 벡터, 선형변환, 시그모이드, 활성화 함수가 어떻게 이어지는지 함께 읽습니다.
3. 실행 후 손실 감소와 정확도를 확인합니다.
4. **주식 AI 실험실**의 뉴런 계산 미니 실습에서 숫자를 바꿔 가중합·편향·시그모이드·오차·비용이 어떻게 달라지는지 확인합니다.
5. 이어서 신경망 모델을 선택해 실제 입력 데이터로 다시 체험합니다.
6. `chapter30`, `chapter101`, `chapter103`으로 이어서 모델 구조를 확장합니다.

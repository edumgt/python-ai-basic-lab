# CHAPTER21 - 신경망 기초와 학습(Forward/Backward)

- 실행: `python practice.py`

---

## 핵심 개념

- **순전파(Forward Pass)**: 입력 데이터가 입력층 → 은닉층 → 출력층 방향으로 흘러가며 예측값을 만드는 과정입니다.  
  예) 주가·거래량·이평선 입력 → 64개 뉴런이 패턴 계산 → 32개 뉴런이 판단 → 상승 확률 출력

- **역전파(Backpropagation)**: 예측이 틀렸을 때 오차를 **출력층 → 은닉층 → 입력층** 방향으로 거꾸로 전달하며 가중치를 수정하는 과정입니다.  
  예) 상승 예측이 틀리면, 출력 뉴런부터 입력 쪽까지 각 뉴런의 가중치를 "이 방향으로 틀렸으니 조금 수정"하며 개선합니다.

- **경사하강법(Gradient Descent)**: 역전파로 계산된 기울기를 이용해 손실이 줄어드는 방향으로 가중치를 조금씩 이동하는 학습 방법입니다.  
  순전파 → 오차 계산 → 역전파 → 가중치 수정의 **피드백 루프**를 수천 번 반복해 모델이 점점 정확해집니다.

## 언제 쓰나요?

- 비선형 패턴이 많아 선형 모델만으로 부족할 때
- 입력 특성이 많고 조합 규칙을 직접 만들기 어려울 때
- CNN, RNN, Transformer 같은 딥러닝 모델의 출발점을 이해할 때

## 결과 해석 체크리스트

- `initial_loss`와 `final_loss` 차이가 충분한지 확인합니다.
- `train_accuracy`가 높아졌다면 실제로 가중치가 학습된 것입니다.
- `weight_shapes`와 `softmax_example`을 보며 층 구조를 해석합니다.

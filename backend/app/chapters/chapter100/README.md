# CHAPTER100 - SVM으로 시장 상태 분류

- 주제: SVM 분류 기초
- 10분 기본 지식: SVM은 경계를 가장 넓게 분리하는 초평면을 찾는 모델입니다.
- 30분 Python 실습: 수익률/변동성 특성으로 상승장/하락장 상태를 분류합니다.
- 실행: `python practice.py`

---

## 핵심 개념

- **마진 최대화**: 두 클래스 사이 여유 공간을 최대한 넓히는 방향을 찾습니다.
- **커널 트릭**: 선형으로 안 나뉘는 데이터도 고차원으로 옮겨 분리할 수 있습니다.
- **분류 경계 해석**: SVM은 적은 수의 핵심 샘플(support vector)에 특히 민감합니다.

## 언제 쓰나요?

- 특성 수가 많고 결정 경계가 복잡한 분류 문제에서
- 작은/중간 크기 데이터셋에서 강한 분류 기준선을 만들고 싶을 때
- 로지스틱 회귀보다 더 단단한 경계를 원할 때

## 결과 해석 체크리스트

- `accuracy`, `f1`을 함께 보고 경계 품질을 확인합니다.
- `train_rows`, `test_rows`로 데이터 분할이 적절한지 확인합니다.
- 커널과 C 값이 바뀌면 결과가 달라진다는 점을 메모합니다.

## 웹앱 실습 순서

1. 챕터 실행으로 SVM 분류 성능을 먼저 확인합니다.
2. 설명 탭에서 마진, support vector, 커널 개념을 정리합니다.
3. 결과 탭에서 `accuracy`와 `f1`을 함께 읽습니다.
4. 이후 로지스틱 회귀(`chapter06`)와 랜덤포레스트(`chapter08`)를 실행해 모델별 경계 차이를 비교합니다.

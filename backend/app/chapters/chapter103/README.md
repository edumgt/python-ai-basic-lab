# CHAPTER103 - Transformer 시계열 기초

- 주제: Transformer 기초
- 10분 기본 지식: Transformer는 attention으로 다음 주가 예측에 중요한 시점을 가중합합니다.
- 30분 Python 실습: 단일 헤드 self-attention으로 다음 거래일 수익률 예측 점수를 계산합니다.
- 실행: `python practice.py`

---

## 핵심 개념

- **Self-Attention**: 모든 시점을 한 번에 보며 중요도를 계산합니다.
- **가중합 표현**: 중요한 시점일수록 더 큰 영향력을 갖습니다.
- **병렬 처리 장점**: 순차 계산 없이 전체 문맥을 동시에 읽을 수 있습니다.

## 언제 쓰나요?

- 긴 시계열에서 멀리 떨어진 구간 관계가 중요할 때
- 여러 특성을 한 번에 보고 중요 시점을 고르고 싶을 때
- 최신 시계열 모델(PatchTST, TFT, iTransformer)로 확장할 때

## 결과 해석 체크리스트

- attention 점수가 어떤 시점에 크게 주어졌는지 확인합니다.
- RNN/LSTM보다 오래전 정보도 직접 비교할 수 있다는 점을 정리합니다.
- 중요도와 예측 결과를 함께 읽는 연습을 합니다.

## 웹앱 실습 순서

1. 설명 탭에서 attention 개념도를 먼저 읽습니다.
2. 실행 후 attention 관련 결과를 확인합니다.
3. `chapter101`, `chapter102`와 함께 시계열 처리 방식 차이를 비교합니다.
4. `chapter112`나 관련 문서에서 멀티 특성 예측 흐름으로 확장합니다.

# CHAPTER113 - 로봇주 vs 반도체주 텍스트 분류

- 주제: TF-IDF + 다중 분류기로 업종 텍스트 자동 분류
- 목표: 종목 설명 텍스트를 보고 로봇 관련주(0)와 반도체 관련주(1)를 자동으로 구분합니다.
- 실행: `python practice.py`

---

## 핵심 개념

- **TF-IDF**: 텍스트를 숫자 벡터로 변환하는 방법입니다. 자주 등장하는 단어보다 특정 문서에서만 많이 나오는 단어에 높은 가중치를 줍니다.
- **문자 n-gram**: 단어 대신 문자 조합(2~4글자)을 특성으로 씁니다. 한국어처럼 형태소 분석이 필요한 언어에서도 잘 작동합니다.
- **다중 분류기 비교**: 로지스틱 회귀, 랜덤 포레스트, SVM 세 모델을 같은 데이터로 비교합니다.

## 왜 이 예제인가요?

`docs/07.md`의 업종 분류 예시(로봇주 vs 반도체주)를 BERT 없이 scikit-learn만으로 체험합니다.  
BERT와 동일한 문제를 경량 모델로 풀어보는 연습입니다.

## 결과 해석 체크리스트

- `model_comparison`에서 세 모델의 `accuracy`와 `f1`을 비교합니다.
- `predictions`가 실제 레이블(로봇주/반도체주)과 얼마나 일치하는지 확인합니다.
- `best_model`이 어떤 알고리즘인지 확인하고 왜 그 모델이 좋은지 생각해봅니다.

# CHAPTER09 - K-Means 군집화

- 주제: K-Means 군집화
- 목표: 주가 예측 전에 비슷한 시장 국면을 그룹으로 묶고 중심을 해석합니다.
- 실행: `python practice.py`

---

## 핵심 개념

- **비지도학습**: 정답 라벨 없이 데이터 구조를 찾는 방식입니다.
- **클러스터 중심**: 각 그룹의 대표 위치를 의미합니다.
- **스케일링 중요성**: 거리 기반 알고리즘이라 특성 크기를 맞춰야 합니다.

## 먼저 구분하면 쉬워요

| 개념 | 쉬운 뜻 | 예시 |
|------|------|------|
| **레이블** | 데이터에 붙은 정답 이름표 | `상승`, `하락`, `보합` |
| **지도학습** | 레이블을 보면서 배우는 방법 | "이 데이터는 상승이야"를 알려주고 학습 |
| **비지도학습** | 레이블 없이 비슷한 것끼리 찾는 방법 | 비슷한 종목끼리 자동 묶기 |
| **군집화** | 비지도학습의 대표 예시 | 성장주 무리, 안정주 무리 찾기 |

이 챕터는 **비지도학습**에 해당합니다.  
즉, 정답을 맞히기보다 **데이터 안에 어떤 무리가 있는지 찾는 연습**입니다.

## 언제 쓰나요?

- 고객·종목·상품을 비슷한 특징끼리 자동 분류하고 싶을 때
- 사전에 정답을 만들기 어렵지만 구조는 찾고 싶을 때
- 분류/회귀 전에 데이터 군집을 먼저 파악하고 싶을 때

## 결과 해석 체크리스트

- 실행 결과의 `cluster_count`가 기대한 K값과 일치하는지 확인합니다.
- 스케일을 바꾸면 군집 결과도 바뀔 수 있음을 기억합니다.
- 군집은 정답이 아니라 **해석을 위한 출발점**이라는 점을 정리합니다.

## 웹앱 실습 순서

1. 설명 탭에서 비지도학습 개념을 먼저 읽습니다.
2. 실행 후 결과 탭에서 군집 개수를 확인합니다.
3. 차트 탭에서 숫자 비교와 함께 `chapter109`의 종목 군집 챕터로 확장 학습합니다.
4. 분류 모델 챕터와 달리 정답 지표가 적다는 점을 메모합니다.

# CHAPTER107 - 백테스트 성과지표 구현

- 주제: MDD/Sharpe/Sortino 계산
- 10분 기본 지식: 백테스트는 수익률뿐 아니라 위험을 함께 측정해야 합니다.
- 30분 Python 실습: 일간 수익률 시퀀스로 핵심 성과지표를 직접 계산합니다.
- 실행: `python practice.py`

---

## 핵심 개념

- **MDD**: 자산이 최고점에서 얼마나 크게 떨어졌는지 보여줍니다.
- **Sharpe/Sortino**: 수익을 위험과 함께 비교하는 대표 지표입니다.
- **백테스트 해석**: 높은 수익률이라도 변동성과 낙폭이 크면 실전 적용이 어렵습니다.

## 언제 쓰나요?

- 모델 예측을 실제 투자 전략으로 연결할 때
- 여러 전략 중 위험 대비 효율이 좋은 쪽을 고를 때
- 단순 정확도보다 실전 운용 품질을 보고 싶을 때

## 결과 해석 체크리스트

- 수익률만 보지 말고 최대 낙폭과 위험 조정 수익을 함께 봅니다.
- 전략이 좋아 보여도 MDD가 너무 크면 운영 난도가 높습니다.
- `chapter10` 지표와 `chapter108` 자산배분을 연결해서 해석합니다.

## 웹앱 실습 순서

1. 챕터 실행 후 성과지표 숫자를 확인합니다.
2. 주식 AI 실험실의 포트폴리오 곡선과 함께 비교해봅니다.
3. 결과 탭에서 위험과 수익을 동시에 읽는 습관을 만듭니다.
4. 이후 포트폴리오 최적화 챕터와 연결합니다.

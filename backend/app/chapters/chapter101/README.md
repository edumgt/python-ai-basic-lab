# CHAPTER101 - RNN 시계열 예측 기초

- 주제: RNN 기초
- 10분 기본 지식: RNN은 이전 시점의 정보를 다음 시점 계산에 전달합니다.
- 30분 Python 실습: 간단한 순환 계산으로 시계열 다음 값 예측을 체험합니다.
- 실행: `python practice.py`

---

## 핵심 개념

- **순환 연결**: 이전 시점의 은닉 상태를 다음 시점으로 넘깁니다.
- **시간 의존성**: 어제의 변화가 오늘 판단에 직접 영향을 줍니다.
- **장기 의존성 한계**: 시퀀스가 길어질수록 오래전 정보가 약해질 수 있습니다.

## 언제 쓰나요?

- 데이터 순서 자체가 중요한 시계열/텍스트 문제에서
- 짧은 구간의 흐름을 기억하며 다음 값을 예측할 때
- LSTM, GRU 같은 발전형 순환 모델로 넘어가기 전에

## 결과 해석 체크리스트

- 시간 순서가 바뀌면 모델 의미가 사라진다는 점을 기억합니다.
- 결과 값이 있다면 최근 구간 정보가 얼마나 반영됐는지 설명해봅니다.
- `chapter102` LSTM과 비교해 장기 기억 한계를 정리합니다.

## 웹앱 실습 순서

1. 설명 탭에서 시퀀스와 은닉 상태 개념을 먼저 읽습니다.
2. 실행 결과로 순환 계산 흐름을 확인합니다.
3. `chapter102`, `chapter103`과 이어서 시계열 모델 진화를 비교합니다.
4. 미니 프로젝트(`chapter112`)에서 시계열 특성 기반 예측으로 연결합니다.

# CHAPTER108 - 포트폴리오 최적화 기초

- 주제: 평균분산·Risk-Parity 포트폴리오
- 10분 기본 지식: 자산배분은 수익률 예측보다 위험 분산이 핵심입니다.
- 30분 Python 실습: 평균분산 가중치와 단순 Risk-Parity 가중치를 비교합니다.
- 실행: `python practice.py`

---

## 핵심 개념

- **자산배분**: 어떤 종목을 얼마나 담을지 정하는 문제입니다.
- **평균-분산 최적화**: 기대수익과 변동성을 함께 고려합니다.
- **Risk Parity**: 자산별 위험 기여도를 비슷하게 맞추는 접근입니다.

## 언제 쓰나요?

- 좋은 종목을 고른 뒤 실제 비중까지 결정해야 할 때
- 한 종목에 몰리지 않고 위험을 분산하고 싶을 때
- 백테스트 성과지표와 연결해 전략을 안정화하고 싶을 때

## 결과 해석 체크리스트

- 높은 기대수익보다 **분산된 위험 구조**가 더 중요한지 확인합니다.
- 상관관계가 높으면 비중을 나눠도 분산 효과가 제한될 수 있습니다.
- MDD/Sharpe와 함께 포트폴리오 관점에서 결과를 해석합니다.

## 웹앱 실습 순서

1. 챕터 실행으로 가중치 비교 결과를 확인합니다.
2. `chapter107`과 함께 위험 지표를 연결해 해석합니다.
3. 예측 실험실에서 여러 종목 결과를 본 뒤 포트폴리오 관점으로 생각해봅니다.
4. 미니 프로젝트 단계에서 단일 모델 결과를 자산배분으로 확장합니다.

# CHAPTER102 - LSTM 시계열 예측 기초

- 주제: LSTM 기초
- 10분 기본 지식: LSTM은 게이트로 기억할 정보와 버릴 정보를 조절합니다.
- 30분 Python 실습: 단순화한 LSTM 셀 계산으로 시계열 패턴 추적을 체험합니다.
- 실행: `python practice.py`

---

## 핵심 개념

- **입력/망각/출력 게이트**: 무엇을 저장하고 지울지 제어합니다.
- **셀 상태**: 장기 기억을 전달하는 핵심 경로입니다.
- **RNN 개선**: 긴 시퀀스에서 정보 손실을 줄이기 위한 구조입니다.

## 언제 쓰나요?

- 과거 여러 시점의 맥락을 함께 기억해야 할 때
- 단순 RNN보다 긴 흐름을 안정적으로 다루고 싶을 때
- 시계열 예측, 자연어, 센서 로그처럼 순서 데이터가 핵심일 때

## 결과 해석 체크리스트

- 게이트가 많아질수록 계산은 복잡하지만 기억력은 좋아진다는 점을 정리합니다.
- RNN보다 긴 패턴을 유지하기 쉽다는 점을 설명해봅니다.
- Transformer와 비교해 순차 처리 vs 전체 주목의 차이를 정리합니다.

## 웹앱 실습 순서

3. `chapter101`과 번갈아 보며 기억 메커니즘 차이를 비교합니다.
4. `chapter103`으로 넘어가 순차 처리와 attention 기반 처리를 연결합니다.

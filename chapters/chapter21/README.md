# CHAPTER21 - 신경망 기초와 학습(Forward/Backward)

- 주제: 신경망 모델 기초
- 목표: 가중치 행렬, 순전파, 손실 계산, 역전파, 경사하강법 업데이트를 한 흐름으로 실습합니다.
- 실행: `python practice.py`

---

## 핵심 개념

- **순전파(Forward)**: 입력 → 은닉층 → 출력으로 예측을 만드는 과정입니다.
- **역전파(Backward)**: 오차를 거꾸로 흘려보내며 가중치를 업데이트합니다.
- **경사하강법**: 손실이 줄어드는 방향으로 조금씩 이동하는 학습 방법입니다.

## 언제 쓰나요?

- 비선형 패턴이 많아 선형 모델만으로 부족할 때
- 입력 특성이 많고 조합 규칙을 직접 만들기 어려울 때
- CNN, RNN, Transformer 같은 딥러닝 모델의 출발점을 이해할 때

## 결과 해석 체크리스트

- `initial_loss`와 `final_loss` 차이가 충분한지 확인합니다.
- `train_accuracy`가 높아졌다면 실제로 가중치가 학습된 것입니다.
- `weight_shapes`와 `softmax_example`을 보며 층 구조를 해석합니다.

## 웹앱 실습 순서

1. 설명 탭에서 순전파/역전파 용어를 먼저 정리합니다.
2. 실행 후 손실 감소와 정확도를 확인합니다.
3. **주식 AI 실험실**에서 신경망 모델을 선택해 실제 입력 데이터로 다시 체험합니다.
4. `chapter30`, `chapter101`, `chapter103`으로 이어서 모델 구조를 확장합니다.

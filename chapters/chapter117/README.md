# Chapter 117: 포스코A&C 주가 예측 — 건설경기·중대재해·건축수주 특성 결합

## 학습 목표

포스코A&C 주가 예측 데이터셋에 **건설경기 지표**, **E&C 중대재해 지표**, **건축 관련 수주 건수**를 추가해
실물 비즈니스 이벤트를 반영한 다중 요인 예측을 수행한다.

## 데이터셋 구성 (2010-01 ~ 2024-12, 월별)

| 구분 | 특성 수 | 주요 항목 |
|------|--------|---------|
| 건설경기/거시 | 5개 | 건설경기 지수, 철강가격 지수, 주택착공, 금리, 3개월 건설경기 평균 |
| 수주 지표 | 4개 | 건축 수주 건수, 토목 수주 건수, 총 수주, 수주 전년비 |
| 안전/리스크 | 4개 | 중대재해 건수, 중대재해 플래그, 3개월 재해 합계, 안전점수 |
| 주가 파생 | 8개 | 전월 종가, 1/3/6개월 수익률, MA3/MA6, MA 괴리, 6개월 변동성 |
| 계절 특성 | 2개 | 월, 분기 |
| **합계** | **23개** | |

- **타겟**: 다음 달 종가 > 이번 달 종가 → 1(상승), 그 외 0(하락)
- **분할**: 시계열 순서 유지 70:30 학습/평가
- **모델**: RandomForest, GradientBoosting, MLP

## 핵심 인사이트

- 건축·토목 수주 증가와 건설경기 개선은 주가 상승 확률을 높이는 요인
- 중대재해 건수 증가는 단기 하방 리스크로 반영
- 안전점수 개선은 재해 리스크 완화 신호로 작동

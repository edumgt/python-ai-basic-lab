# Chapter 115: 롯데호텔 주가 예측 — 가상 데이터셋 생성 및 ML/DL 예측

## 학습 목표

30개 체인 호텔 객실예약률·계절 특성·10년 월말 주가 변동을 담은 **가상 1000행 데이터셋**을 생성하고,
ML/DL 다양한 모델로 다음 달 롯데호텔 주가의 상승/하락을 예측한다.

## 데이터셋 구성

| 구분 | 열 수 | 설명 |
|------|-------|------|
| 체인 호텔 예약률 | 30개 | hotel_01_occ ~ hotel_30_occ (%) |
| 계절 특성 | 4개 | month, quarter, season, is_peak_season |
| 주가 파생 특성 | 7개 | prev_month_close, prev_3m/6m/12m_return, price_ma3/ma6, volatility_6m |
| **합계** | **41개** | 전체 특성 수 |

- **총 샘플**: 1000개 (월별)
- **학습/시험**: 700 / 300 분할 (70:30)
- **타겟**: 다음 달 종가 > 이번 달 종가 → 1(상승), 그 외 → 0(하락)

## 웹앱 확인

브라우저에서 `/hotel-stock` 경로로 이동하면 ML/DL 모델 선택 예측 웹앱을 사용할 수 있습니다.

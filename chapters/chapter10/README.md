# CHAPTER10 - 모델 평가 지표

- 주제: 모델 평가 지표
- 목표: 정확도·정밀도·재현율·F1·ROC-AUC를 비교합니다.
- 실행: `python practice.py`

---

## 핵심 개념

- **정확도**: 전체 예측 중 맞춘 비율입니다.
- **정밀도/재현율**: 양성 예측을 얼마나 조심스럽고 놓치지 않게 했는지 보여줍니다.
- **ROC-AUC**: 임계값을 바꿔도 모델이 순위를 잘 매기는지 평가합니다.

## 언제 쓰나요?

- 모델끼리 성능을 공정하게 비교할 때
- 클래스 불균형이 있어 정확도 하나만 보면 위험할 때
- 실전 손실/기회비용을 생각해 지표를 함께 봐야 할 때

## 결과 해석 체크리스트

- 실행 결과의 `precision`, `recall`, `roc_auc`를 함께 봅니다.
- 지표 간 충돌이 생기면 비즈니스 목표를 먼저 정해야 합니다.
- 투자 문제에서는 정확도보다 정밀도와 수익률이 더 중요할 수 있습니다.

## 웹앱 실습 순서

1. 챕터 실행으로 주요 지표 값을 먼저 확인합니다.
2. 설명 탭에서 각 지표가 어떤 상황에 적합한지 정리합니다.
3. **주식 AI 실험실** 또는 **예측 실험실**로 이동해 동일한 지표가 실제 UI에 어떻게 표시되는지 확인합니다.
4. 모델을 바꾸며 지표가 어떻게 달라지는지 비교합니다.

# Chapter 116: 현대자동차 주가 예측 — 생산·판매·전기차 특성 기반 ML/DL 고도화

## 학습 목표

현대자동차의 **년도별 생산차량수**, **미국 판매량**, **전기차(EV) 생산 비율** 등 실물 비즈니스 특성을
주가 기술적 지표와 결합한 **가상 월별 데이터셋(2010–2024)**을 구축하고,
ML(RandomForest·GradientBoosting)과 DL(MLP 신경망)로 다음 달 주가 상승 여부를 예측·비교한다.

## 데이터셋 구성 (2010-01 ~ 2024-12, 180개월)

| 구분 | 특성 수 | 주요 항목 |
|------|--------|---------|
| 생산 지표 | 6개 | 전체·국내·해외·EV 월별 생산량, EV 비율(%), EV 비율 전년비(pp) |
| 판매 지표 | 5개 | 미국·중국·유럽·국내 월별 판매량, 미국 판매 전년비(%) |
| 거시 지표 | 3개 | 환율(원/달러), 유가(달러/배럴), 글로벌 자동차 수요 지수 |
| 계절 특성 | 4개 | 월, 분기, 여름 더미, 겨울 더미 |
| 주가 파생 | 10개 | 전월 종가, 1/3/6/12개월 수익률, 단기/중기/장기 이동평균, MA괴리, 변동성 |
| **합계** | **28개** | |

- **총 샘플**: ~168개 (결측 제거 후)
- **학습/시험**: 70:30 시계열 순서 분리 (117 / 51)
- **타겟**: 다음 달 종가 > 이번 달 종가 → 1(상승), 그 외 → 0(하락)

## 가상 데이터 시뮬레이션 설계

| 이벤트 | 반영 내용 |
|--------|---------|
| 유가 급락(2014~16) | 유가 × 0.60 |
| 코로나(2020.3~2021.6) | 생산·판매 × 0.65~0.75 |
| 우크라이나 전쟁(2022~) | 환율 +80원, 유가 × 1.35 |
| 중국 시장 둔화 | 중국 판매 선형 감소(70만→25만대/년) |
| EV 전환 가속 | EV 비율 0.2%(2010) → 25%(2024) |

## 모델 비교

| 모델 | 유형 | 구조/파라미터 |
|------|------|-------------|
| **RandomForest** | ML | n_estimators=200, max_depth=6 |
| **GradientBoosting** | ML | n_estimators=200, max_depth=4, lr=0.05 |
| **MLP Neural Network** | DL | 3층(64→32→16), ReLU, Adam, Early Stopping |

## 핵심 인사이트

- **EV 비율 전년비(ev_ratio_12m_chg)** 와 **미국 판매 전년비(us_sales_12m_chg)** 가 상위 특성에 포함됨
- 환율(원/달러)은 수출 비중이 큰 현대차 주가에 강한 역방향 영향
- DL(MLP)은 EV 전환 초기처럼 비선형 구조 변화 시 ML 앙상블을 보완
- GradientBoosting이 AUC 기준 최우수 모델로 선정됨

## 실습 확장 아이디어

1. **회귀 예측**: 분류(상승/하락) 대신 다음 달 종가 수준 예측
2. **LSTM**: 시계열 순서 의존성을 학습하는 순환 신경망 적용
3. **실데이터 연동**: FinanceDataReader로 005380(현대차) 실주가 조회
4. **멀티 팩터**: R&D 투자, 수소차(FCEV) 개발 진행률, 충전 인프라 보급률 추가

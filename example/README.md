# example 실습 파이썬 파일 재배치 안내

`docs/01.md`~`docs/12.md`의 12일 학습 흐름 순서에 맞춰, `example` 아래 실습용 `.py` 파일을 `ch01`~`ch12`로 재배치했습니다.

- `ch01` 기초 연산/텐서 감각
- `ch02` 데이터프레임·전처리 기초
- `ch03` 지도학습 입문
- `ch04` 검증/분류 확장
- `ch05` 데이터 준비·허브형 기본 실습
- `ch06` 신경망 기초 실습
- `ch07` 텍스트/NLP 기초
- `ch08` 시퀀스·딥러닝 NLP
- `ch09` 챗봇/웹 연동 스크립트
- `ch10` OCR 실습
- `ch11` 이미지 분류 실습
- `ch12` 비전 응용/유사도 실습

참고:
- `docs/13.md`는 LLM 파인튜닝 확장 주제이므로 이번 `ch01`~`ch12` 재배치 범위에서 제외했습니다.
- `.py` 외의 데이터/이미지/모델 파일은 기존 위치를 유지했습니다.

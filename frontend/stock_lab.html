실데이터 기반 주식 예측 실험

복잡한 메뉴 대신, 같은 데이터셋으로 ML과 DL 모델을 비교하고 결과를 쉬운 말로 해석하는 흐름만 남겼습니다.

직접 입력 OHLCV

1. 데이터 선택

샘플 종목, 실제 CSV, 내장 데이터셋 중 하나를 불러옵니다.

2. 모델 실행

비교 모드면 ML/DL 4개 모델을 한 번에 돌리고 추천 모델을 뽑습니다.

3. 결과 해석

정확도, AUC, 수익률, 중요 특성을 사람 읽기 쉬운 설명으로 정리합니다.

📋 주가 데이터 입력

🗂 데이터셋 허브

날짜 · 시가 · 고가 · 저가 · 종가(원) · 거래량을 입력하세요. 최소 30행 권장

KRX yfinance 티커 규칙
코스피: 종목코드.KS (예: 005930.KS)
코스닥: 종목코드.KQ (예: 086520.KQ)
CSV 업로드 헤더는 date,open,high,low,close,volume 형식이 가장 좋고, date,close,volume도 자동 보정됩니다.

🤖 예측 모델 선택

단일 모델 상세 보기용 선택입니다. 비교 모드에서는 4개 모델을 모두 실행합니다.

ML 3개 + DL 1개

🧪 실행 방식

비교 모드는 여러 ML/DL 모델의 성향 차이를 보여주고, 가장 균형 잡힌 모델을 자동으로 상세 결과로 펼쳐줍니다.

학습 80% · 테스트 20%로 분할해 모델을 평가합니다

실데이터셋으로 ML · DL 주식 예측을 단순한 흐름으로 체험하는 웹앱

FastAPI 백엔드가 OHLCV·거시경제·DART 공시 데이터를 처리하고, 로지스틱 회귀부터 Transformer까지 여러 모델 결과를 브라우저에서 바로 비교합니다. Ollama AI가 숫자 결과를 쉬운 말로 해설해 줍니다.

📈 추천 시작점

실데이터 모델 비교

`stock_ohlcv.csv`, `traffic_timeseries.csv`, 직접 업로드 CSV를 같은 화면에서 비교합니다. 로지스틱 회귀, 랜덤 포레스트, 그래디언트 부스팅, 신경망 결과를 한 번에 볼 수 있습니다.

실데이터 선택 → 전체 모델 비교 → 추천 모델 상세 해석
🗂 데이터 먼저 보기

데이터셋 허브

`data/` 폴더의 CSV를 미리보기와 자동 차트로 확인합니다. 어떤 데이터가 시계열 예측용인지, 어떤 데이터가 분류나 군집화용인지 빠르게 구분할 수 있습니다.

CSV 구조 파악 → 실험 화면으로 이동
🎯 업로드 중심

파일 업로드형 예측

여러 회사가 들어 있는 CSV를 올려 타겟 예측을 수행합니다. 모델별 특성 중요도와 회사별 결과를 보고 싶을 때 사용하는 보조 화면입니다.

기업별 CSV 예측과 특성 비교
🧭 뉴스 해석

이벤트 투자 컨설팅

전쟁, 가뭄, 물가, 금리, 규제 같은 뉴스 문장을 넣으면 시장 체온, 수혜/피해 업종, 관심 종목 메모를 컨설팅처럼 정리해 줍니다.

뉴스 입력 → 업종 영향도 → 체크리스트 확인
📑 공시 기반

DART 공시 투자 파이프라인

DART 재무제표, 최근 공시, 가격 팩터를 묶어 회사 체력과 투자 관찰 점수를 보여줍니다. 숫자만 보지 않고 왜 그런 판단이 나왔는지 쉬운 말로 읽을 수 있습니다.

공시 데이터셋 생성 → 회사 선택 → 투자 해석 확인
🌍 거시 확장

거시경제 투자 파이프라인

FRED와 World Bank에서 모은 금리, 물가, 실업률, VIX 같은 외부 데이터를 DART 재무와 합쳐 ML/DL 모델로 다음 해 실적 개선을 분류합니다.

외부 API 추천 → 특징 생성 → ML/DL 비교
💳 대출 투자 분석

마이너스 통장 투자 분석

2,000만 원 마이너스 통장 자금을 주식 투자할 때 이익인지 시뮬레이션합니다. 금리·세금·변동성·심리적 손절선 등 다변수 분석과 몬테카를로 시뮬레이션으로 성공 확률을 추정합니다.

변수 입력 → 몬테카를로 시뮬레이션 → 투자 판정
🤖 LLM 파인튜닝

LLM 파인튜닝 실험실

LoRA·QLoRA·Instruction Tuning 기법을 소개하고, 파라미터 압축 효과와 도메인 특화 파인튜닝 전후 성능 차이를 직접 수치로 확인합니다.

LoRA 시뮬레이터 → Instruction Tuning → 실전 로드맵
🧬 벡터 DB 탐색

Qdrant 벡터 뷰어

Docker로 띄운 Qdrant의 컬렉션과 포인트 벡터를 웹에서 직접 확인합니다. 포인트 개수, 벡터 차원, payload를 빠르게 점검할 수 있습니다.

컬렉션 조회 → 포인트 샘플 확인 → 벡터 구조 점검

추천 사용 순서

처음에는 /lab에서 내장 실데이터셋을 불러 전체 모델 비교를 실행하는 흐름이 가장 이해하기 쉽습니다. 이후 /dart·/macro로 확장하세요.

이 앱이 하는 일

FastAPI 백엔드가 ML/DL 숫자 결과를 만들고, Ollama가 그 결과를 자연어로 해설합니다. DART·FRED·World Bank 데이터까지 한 화면에서 투자 관점으로 읽을 수 있습니다.

읽어야 할 지표

AUC는 분류력, 전략 수익률은 투자 적용 감각, 중요 특성은 모델이 어떤 신호를 봤는지 알려줍니다. ICIR·Sharpe Ratio로 신호 일관성을 추가 확인하세요.

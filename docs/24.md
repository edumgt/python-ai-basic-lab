## 1. Transformer 등장 배경

RNN/LSTM의 한계:
- **순차 처리**: 병렬화 불가 → 학습 속도 느림
- **고정 크기 병목**: 모든 정보를 은닉 상태 하나에 압축
- **장거리 의존성**: 거리가 멀수록 정보 소실

Transformer의 해결책: **Self-Attention** — 모든 시점이 서로 직접 소통

---

## 2. Self-Attention 메커니즘

$$\text{Attention}(Q, K, V) = \text{softmax}\left(\frac{QK^T}{\sqrt{d_k}}\right) V$$

- **Q (Query)**: "지금 무엇을 찾고 있나?"
- **K (Key)**: "내가 가진 정보는 무엇인가?"
- **V (Value)**: "실제로 전달할 내용"
- $\sqrt{d_k}$ **스케일링**: 내적값이 커져 softmax가 포화되는 것 방지

```
시계열: [x₁, x₂, x₃, ..., x_T]

각 위치 t에서 다른 모든 위치와 직접 상관관계 계산:
      t=1   t=2   t=3   t=4
t=1 [0.1  0.6   0.2   0.1]  ← t=1이 각 위치에 주목하는 가중치
t=2 [0.3  0.1   0.5   0.1]
t=3 [0.2  0.4   0.2   0.2]
t=4 [0.1  0.2   0.3   0.4]
```

### Multi-Head Attention

여러 Attention 헤드가 다른 "관점"으로 패턴 학습:

$$\text{MultiHead}(Q, K, V) = \text{Concat}(\text{head}_1, ..., \text{head}_h)W^O$$
$$\text{head}_i = \text{Attention}(QW_i^Q, KW_i^K, VW_i^V)$$

## 3. Positional Encoding

Transformer는 순서 정보가 없어 **위치 인코딩**을 입력에 추가합니다.

$$PE_{(pos, 2i)} = \sin\left(\frac{pos}{10000^{2i/d_{model}}}\right)$$
$$PE_{(pos, 2i+1)} = \cos\left(\frac{pos}{10000^{2i/d_{model}}}\right)$$

```python
import numpy as np
import torch
import torch.nn as nn
import matplotlib.pyplot as plt

def positional_encoding(seq_len: int, d_model: int) -> torch.Tensor:
    pe = torch.zeros(seq_len, d_model)
    pos = torch.arange(seq_len).unsqueeze(1).float()
    div = torch.exp(torch.arange(0, d_model, 2).float() * (-np.log(10000.0) / d_model))
    pe[:, 0::2] = torch.sin(pos * div)
    pe[:, 1::2] = torch.cos(pos * div)
    return pe

pe = positional_encoding(50, 64)

plt.figure(figsize=(10, 4))
plt.imshow(pe.numpy().T, aspect='auto', cmap='RdBu')
plt.colorbar()
plt.xlabel('시퀀스 위치')
plt.ylabel('차원')
plt.title('Sinusoidal Positional Encoding')
plt.tight_layout()
plt.savefig('positional_encoding.png', dpi=120)
print("저장: positional_encoding.png")
```

## 4. Transformer Encoder 구조

```
입력 임베딩 + Positional Encoding
        ↓
  ┌─────────────────────────────┐
  │  Multi-Head Self-Attention  │
  │  + Residual Connection      │
  │  + Layer Normalization      │
  │                             │
  │  Feed-Forward Network       │
  │  (Linear → ReLU → Linear)   │
  │  + Residual Connection      │
  │  + Layer Normalization      │
  └─────────────────────────────┘ × N layers
        ↓
     출력 (각 시점의 컨텍스트 벡터)
```

## 5. 실습: Transformer로 주가 방향 예측

```python
import torch
import torch.nn as nn
import numpy as np
import pandas as pd
from torch.utils.data import DataLoader, TensorDataset
from sklearn.preprocessing import StandardScaler
from sklearn.metrics import roc_auc_score

np.random.seed(42)
torch.manual_seed(42)

# 데이터 생성 (Day 037과 동일한 방식)
n_days = 1500
price = 100 * np.exp((np.random.randn(n_days) * 0.01).cumsum())
s = pd.Series(price)
ret = s.pct_change().fillna(0)

feat = pd.DataFrame({
    'ret_1':   ret,
    'ret_5':   ret.rolling(5).sum(),
    'ret_20':  ret.rolling(20).sum(),
    'vol_20':  ret.rolling(20).std(),
    'rsi':     (ret.clip(lower=0).rolling(14).mean() /
                ret.abs().rolling(14).mean()),
}).fillna(0).values.astype(np.float32)

label = (ret.shift(-1) > 0).astype(int).values  # 다음날 상승?

SEQ_LEN = 30
X_list, y_list = [], []
for i in range(len(feat) - SEQ_LEN):
    X_list.append(feat[i:i+SEQ_LEN])
    y_list.append(label[i+SEQ_LEN])
X_arr = np.array(X_list)
y_arr = np.array(y_list, dtype=np.float32)

scaler = StandardScaler()
X_arr_2d = X_arr.reshape(-1, X_arr.shape[-1])
X_arr = scaler.fit_transform(X_arr_2d).reshape(X_arr.shape).astype(np.float32)

split_tr  = int(len(X_arr) * 0.7)
split_val = int(len(X_arr) * 0.85)

X_tr_t  = torch.tensor(X_arr[:split_tr])
X_val_t = torch.tensor(X_arr[split_tr:split_val])
X_te_t  = torch.tensor(X_arr[split_val:])
y_tr_t  = torch.tensor(y_arr[:split_tr])
y_val_t = torch.tensor(y_arr[split_tr:split_val])
y_te_np = y_arr[split_val:]

train_loader = DataLoader(TensorDataset(X_tr_t, y_tr_t), batch_size=64, shuffle=True)
val_loader   = DataLoader(TensorDataset(X_val_t, y_val_t), batch_size=256)
```

### Transformer 모델 정의

```python
class QuantTransformer(nn.Module):
    """
    시계열 분류용 Transformer Encoder
    입력: (batch, seq_len, n_features)
    출력: (batch,) — 이진 로짓
    """
    def __init__(self, n_features: int, d_model: int = 64, nhead: int = 4,
                 num_layers: int = 2, dim_ff: int = 256, dropout: float = 0.1):
        super().__init__()
        self.input_proj = nn.Linear(n_features, d_model)
        self.pe_dropout = nn.Dropout(dropout)

        encoder_layer = nn.TransformerEncoderLayer(
            d_model=d_model,
            nhead=nhead,
            dim_feedforward=dim_ff,
            dropout=dropout,
            batch_first=True,   # (batch, seq, d_model)
            norm_first=True,    # Pre-LN: 더 안정적인 학습
        )
        self.transformer = nn.TransformerEncoder(encoder_layer, num_layers=num_layers)
        self.classifier = nn.Sequential(
            nn.Linear(d_model, 32),
            nn.ReLU(),
            nn.Dropout(dropout),
            nn.Linear(32, 1),
        )

        # Positional Encoding 등록
        self.register_buffer('pe', positional_encoding(500, d_model))

    def forward(self, x):
        # x: (batch, seq_len, n_features)
        seq_len = x.size(1)
        x = self.input_proj(x) + self.pe[:seq_len]  # 입력 투영 + 위치 인코딩
        x = self.pe_dropout(x)
        x = self.transformer(x)          # (batch, seq, d_model)
        x = x.mean(dim=1)               # 시간 축 평균 풀링
        return self.classifier(x).squeeze(-1)

model = QuantTransformer(n_features=5, d_model=64, nhead=4, num_layers=2)
print(f"파라미터 수: {sum(p.numel() for p in model.parameters()):,}")

# 입출력 형태 확인
dummy = torch.randn(4, SEQ_LEN, 5)
print(f"입력: {dummy.shape} → 출력: {model(dummy).shape}")
```

### 훈련

```python
optimizer = torch.optim.AdamW(model.parameters(), lr=1e-3, weight_decay=1e-2)
scheduler = torch.optim.lr_scheduler.OneCycleLR(
    optimizer, max_lr=1e-3, epochs=60,
    steps_per_epoch=len(train_loader),
)
criterion = nn.BCEWithLogitsLoss()

best_auc, best_w = 0, None

for epoch in range(1, 61):
    model.train()
    for X_b, y_b in train_loader:
        logits = model(X_b)
        loss   = criterion(logits, y_b)
        optimizer.zero_grad(); loss.backward()
        nn.utils.clip_grad_norm_(model.parameters(), 1.0)
        optimizer.step(); scheduler.step()

    model.eval()
    with torch.no_grad():
        val_prob = torch.sigmoid(model(X_val_t)).numpy()
    val_auc = roc_auc_score(y_val_t.numpy(), val_prob)

    if val_auc > best_auc:
        best_auc = val_auc
        best_w = {k: v.clone() for k, v in model.state_dict().items()}

    if epoch % 10 == 0:
        print(f"Epoch {epoch:3d} | Val AUC={val_auc:.4f} | Best={best_auc:.4f}")

model.load_state_dict(best_w)
model.eval()
with torch.no_grad():
    te_prob = torch.sigmoid(model(X_te_t)).numpy()
print(f"\n테스트 AUC: {roc_auc_score(y_te_np, te_prob):.4f}")
```

## 6. Attention 가중치 시각화

```python
def get_attention_weights(model, x_input):
    """마지막 Encoder Layer의 Self-Attention 가중치 추출"""
    weights = {}
    hooks = []

    def hook_fn(layer_idx):
        def fn(module, inp, out):
            weights[layer_idx] = out[1]  # (batch, heads, seq, seq)
        return fn

    for i, layer in enumerate(model.transformer.layers):
        h = layer.self_attn.register_forward_hook(hook_fn(i))
        hooks.append(h)

    model.eval()
    with torch.no_grad():
        model(x_input)

    for h in hooks:
        h.remove()

    return weights

# 첫 번째 샘플 분석
sample = X_te_t[:1]
attn_weights = get_attention_weights(model, sample)

# 마지막 레이어, 첫 헤드
last_layer_key = max(attn_weights.keys())
if attn_weights[last_layer_key] is not None:
    w = attn_weights[last_layer_key][0, 0].numpy()  # (seq, seq)
    plt.figure(figsize=(8, 6))
    plt.imshow(w, cmap='Blues', vmin=0)
    plt.colorbar()
    plt.xlabel('Key (출처 시점)')
    plt.ylabel('Query (현재 시점)')
    plt.title(f'Self-Attention Map (Layer {last_layer_key}, Head 0)')
    plt.tight_layout()
    plt.savefig('attention_map.png', dpi=120)
    print("저장: attention_map.png")
```

## 핵심 요약

| 구성 요소 | 역할 | 핵심 파라미터 |
|-----------|------|-------------|
| Self-Attention | 모든 시점 간 직접 상관관계 | `d_model`, `nhead` |
| Positional Encoding | 순서 정보 주입 | 입력에 더하기 |
| Feed-Forward | 비선형 변환 | `dim_feedforward` |
| Layer Norm (Pre-LN) | 학습 안정화 | `norm_first=True` 권장 |
| AdamW | 가중치 감쇠 포함 최적화 | `weight_decay=1e-2` |

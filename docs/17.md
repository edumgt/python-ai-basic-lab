## 1. SVM 핵심 원리

SVM은 두 클래스를 분리하는 **결정 초평면(hyperplane)** 중에서 **마진(margin)이 최대인 것**을 찾습니다.

```
클래스 A (+)                   클래스 B (−)
    ●  ●                          ○  ○
      ●   ← 마진 →  결정경계  ← 마진 →  ○
    ●  ●                          ○  ○
  서포트 벡터                   서포트 벡터
```

$$\max_{\mathbf{w}, b} \frac{2}{\|\mathbf{w}\|} \quad \text{s.t.} \quad y_i(\mathbf{w}^T\mathbf{x}_i + b) \ge 1$$

- **서포트 벡터(Support Vector)**: 결정 경계에 가장 가까운 샘플들. 이 점들만이 초평면을 결정함.
- **마진(Margin)**: 두 클래스 서포트 벡터 사이의 거리. 클수록 일반화 성능 높음.
- **C 파라미터**: 마진과 오분류 허용의 트레이드오프 제어.

### 소프트 마진 (Soft Margin)

실제 데이터는 완전 분리가 불가능할 때가 많아 슬랙 변수 $\xi_i$를 도입:

$$\min_{\mathbf{w}, b, \boldsymbol{\xi}} \frac{1}{2}\|\mathbf{w}\|^2 + C\sum_i \xi_i \quad \text{s.t.} \quad y_i(\mathbf{w}^T\mathbf{x}_i+b) \ge 1-\xi_i, \; \xi_i \ge 0$$

| C 값 | 마진 | 오분류 허용 | 적합 상황 |
|------|------|-----------|----------|
| 작은 C (0.01~0.1) | 넓음 | 많이 허용 | 노이즈 많은 금융 데이터 |
| 큰 C (10~100) | 좁음 | 거의 없음 | 선명한 패턴이 있을 때 |

## 2. 커널 트릭 (Kernel Trick)

비선형 데이터를 고차원으로 변환해 선형 분리하는 기법.  
실제 변환을 하지 않고 커널 함수 $K(\mathbf{x}_i, \mathbf{x}_j)$만 계산합니다.

| 커널 | 함수 | 특징 |
|------|------|------|
| Linear | $\mathbf{x}_i^T\mathbf{x}_j$ | 빠름, 고차원에 적합 |
| RBF (Gaussian) | $\exp(-\gamma\|\mathbf{x}_i-\mathbf{x}_j\|^2)$ | 가장 범용적, `gamma` 조정 필요 |
| Polynomial | $(\mathbf{x}_i^T\mathbf{x}_j + r)^d$ | 특정 다항식 패턴에 유효 |

**gamma**: RBF 커널의 영향 반경.  
- 큰 gamma → 좁은 영향 반경 → 결정경계가 복잡해짐 (과적합 위험)
- 작은 gamma → 넓은 영향 반경 → 결정경계가 부드러워짐 (과소적합 위험)

## 3. 실습: SVM으로 주가 방향 예측

```python
import numpy as np
import pandas as pd
from sklearn.svm import SVC, SVR
from sklearn.preprocessing import StandardScaler
from sklearn.metrics import classification_report, roc_auc_score
from sklearn.model_selection import GridSearchCV, TimeSeriesSplit

np.random.seed(42)
n = 800

# 기술적 팩터 시뮬레이션
close   = 100 * np.cumprod(1 + np.random.randn(n) * 0.01)
returns = np.diff(np.log(close))

# 팩터 계산
def rolling_zscore(x, window=20):
    series = pd.Series(x)
    return ((series - series.rolling(window).mean()) / series.rolling(window).std()).values

momentum_5  = rolling_zscore(np.concatenate([[0]*5, returns[:-5].cumsum()]))
momentum_20 = rolling_zscore(np.concatenate([[0]*20, returns[:-20].cumsum()]))
volatility  = rolling_zscore(pd.Series(returns).rolling(20).std().values)
rsi_signal  = rolling_zscore(np.where(returns > 0, returns, 0))  # 단순화된 RSI 대리

# 레이블: 다음 5일 수익률 방향
fwd_ret = np.array([returns[i:i+5].sum() if i+5 < len(returns) else 0 for i in range(len(returns))])
y = (fwd_ret > 0).astype(int)

# 데이터 정렬 (팩터 지연 주의)
X_raw = np.column_stack([momentum_5, momentum_20, volatility, rsi_signal])
valid = ~np.isnan(X_raw).any(axis=1)
X_raw = X_raw[valid]
y = y[valid]

scaler = StandardScaler()
X = scaler.fit_transform(X_raw)

# 시계열 분할 (미래 데이터 누설 방지)
split = int(len(X) * 0.8)
X_tr, X_te = X[:split], X[split:]
y_tr, y_te = y[:split], y[split:]

print(f"훈련 샘플: {len(X_tr)}, 테스트 샘플: {len(X_te)}")
print(f"상승 비율: {y_te.mean():.2%}")
```

### SVM 모델 비교

```python
models = {
    'SVM-Linear': SVC(kernel='linear', C=1.0, probability=True, random_state=42),
    'SVM-RBF':    SVC(kernel='rbf', C=1.0, gamma='scale', probability=True, random_state=42),
    'SVM-Poly':   SVC(kernel='poly', degree=3, C=1.0, probability=True, random_state=42),
}

for name, model in models.items():
    model.fit(X_tr, y_tr)
    pred  = model.predict(X_te)
    prob  = model.predict_proba(X_te)[:, 1]
    auc   = roc_auc_score(y_te, prob)
    acc   = (pred == y_te).mean()
    n_sv  = model.n_support_.sum()  # 서포트 벡터 수
    print(f"\n{name}: Acc={acc:.4f}, AUC={auc:.4f}, 서포트벡터수={n_sv}")
```

### GridSearchCV로 하이퍼파라미터 탐색

```python
from sklearn.model_selection import GridSearchCV

param_grid = {
    'C':     [0.1, 1.0, 10.0, 100.0],
    'gamma': ['scale', 'auto', 0.01, 0.001],
}

tscv = TimeSeriesSplit(n_splits=5)
grid_search = GridSearchCV(
    SVC(kernel='rbf', probability=True, random_state=42),
    param_grid=param_grid,
    cv=tscv,
    scoring='roc_auc',
    n_jobs=-1,
    verbose=1,
)
grid_search.fit(X_tr, y_tr)

print(f"\n최적 파라미터: {grid_search.best_params_}")
print(f"CV AUC: {grid_search.best_score_:.4f}")

best_svm = grid_search.best_estimator_
y_prob_best = best_svm.predict_proba(X_te)[:, 1]
print(f"테스트 AUC: {roc_auc_score(y_te, y_prob_best):.4f}")
```

## 4. SVR: 주가 수익률 회귀 예측

```python
from sklearn.svm import SVR
from sklearn.metrics import mean_squared_error

# 연속값 예측 (회귀)
svr = SVR(kernel='rbf', C=10.0, epsilon=0.01, gamma='scale')
svr.fit(X_tr, fwd_ret[:split][valid[:split]])

y_reg_te = fwd_ret[split:][valid[split:]]
y_reg_pred = svr.predict(X_te)

rmse = np.sqrt(mean_squared_error(y_reg_te, y_reg_pred))
ic   = np.corrcoef(y_reg_te, y_reg_pred)[0, 1]
print(f"SVR RMSE: {rmse:.5f}")
print(f"SVR IC (피어슨): {ic:.4f}")

# 퀀트 핵심 지표: Rank IC (스피어만)
from scipy.stats import spearmanr
rank_ic, _ = spearmanr(y_reg_te, y_reg_pred)
print(f"SVR Rank IC: {rank_ic:.4f}")
```

## 5. SVM의 장단점 및 퀀트 활용 가이드

### 장점
- 고차원 특성 공간에서도 효과적 (팩터 수 > 샘플 수일 때)
- 커널 트릭으로 비선형 패턴 학습
- 마진 최대화로 과적합에 상대적으로 강함

### 단점
- 대용량 데이터에서 학습 속도 느림 (O(n²) ~ O(n³))
- 확률 출력이 기본 제공되지 않음 (`probability=True` 시 Platt scaling 사용, 느림)
- 특성 스케일에 민감 → **StandardScaler 전처리 필수**

### 퀀트 실전 팁

```python
# 팁 1: SVM 신호 강도를 포지션 사이징에 활용
prob_up = best_svm.predict_proba(X_te)[:, 1]  # 상승 확률
position = np.where(prob_up > 0.6, 1.0,        # 강한 매수
          np.where(prob_up < 0.4, -1.0, 0.0)) # 강한 매도 / 관망

# 팁 2: 시장 국면별 SVM 앙상블

```

## 핵심 요약

- **C**: 클수록 훈련 오차 감소 (과적합), 작을수록 마진 넓음 (과소적합)
- **gamma (RBF)**: 클수록 복잡한 경계, 작을수록 단순한 경계
- **커널 선택**: 금융 데이터는 RBF가 일반적으로 무난, Linear는 팩터 많을 때 유리
- **전처리**: StandardScaler 없이는 SVM이 제대로 동작하지 않음

## 실습 과제

```python
# 과제: C와 gamma 조합 히트맵 시각화
# 1) C: [0.01, 0.1, 1, 10, 100], gamma: [0.001, 0.01, 0.1, 1.0, 'scale'] 격자 탐색
# 2) 각 조합의 TimeSeriesSplit CV AUC를 2D 히트맵으로 시각화
# 3) 최적 조합과 과적합 구간을 표시

import matplotlib.pyplot as plt
import seaborn as sns

C_range     = [0.01, 0.1, 1, 10, 100]
gamma_range = [0.001, 0.01, 0.1, 1.0]
scores_mat  = np.zeros((len(C_range), len(gamma_range)))

tscv = TimeSeriesSplit(n_splits=3)
for i, c in enumerate(C_range):
    for j, g in enumerate(gamma_range):
        model = SVC(kernel='rbf', C=c, gamma=g, probability=True)
        fold_scores = []
        for tr_idx, val_idx in tscv.split(X_tr):
            model.fit(X_tr[tr_idx], y_tr[tr_idx])
            prob = model.predict_proba(X_tr[val_idx])[:, 1]
            fold_scores.append(roc_auc_score(y_tr[val_idx], prob))
        scores_mat[i, j] = np.mean(fold_scores)

plt.figure(figsize=(8, 5))
sns.heatmap(scores_mat, xticklabels=gamma_range, yticklabels=C_range,
            annot=True, fmt='.3f', cmap='RdYlGn')
plt.xlabel('gamma')
plt.ylabel('C')
plt.title('SVM RBF 하이퍼파라미터 AUC 히트맵')
plt.tight_layout()
plt.savefig('svm_heatmap.png', dpi=120)
print("저장: svm_heatmap.png")
```

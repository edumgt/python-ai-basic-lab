## 1. 클러스터링 개요

레이블(정답) 없이 데이터의 **내재적 구조**를 발견하는 비지도 학습.

| 알고리즘 | 군집 형태 | 파라미터 | 이상치 | 퀀트 적용 |
|---------|---------|---------|-------|---------|
| **K-Means** | 구형 | K (사전 지정) | 민감 | 종목 그룹화 |
| **DBSCAN** | 임의 형태 | ε, min_samples | 자동 탐지 | 이상 거래 탐지 |
| **계층적** | 덴드로그램 | 연결 방식 | 민감 | 시장 구조 분석 |
| **GMM** | 타원형 | K (사전 지정) | 확률적 | 레짐 분류 |

## 2. K-Means 클러스터링

### 알고리즘

1. K개의 중심점(centroid) 랜덤 초기화 (K-Means++ 사용 권장)
2. 각 샘플을 가장 가까운 중심에 배정
3. 배정된 샘플들의 평균으로 중심 이동
4. 수렴할 때까지 2-3 반복

$$\min_{C_1,...,C_K} \sum_{k=1}^{K}\sum_{\mathbf{x}\in C_k} \|\mathbf{x} - \boldsymbol{\mu}_k\|^2 \quad \text{(Inertia 최소화)}$$

### 실습: 주식 종목 군집화

```python
import numpy as np
import pandas as pd
from sklearn.cluster import KMeans, DBSCAN, AgglomerativeClustering
from sklearn.preprocessing import StandardScaler
from sklearn.metrics import silhouette_score, calinski_harabasz_score
from sklearn.decomposition import PCA
import matplotlib.pyplot as plt

np.random.seed(42)
n_stocks = 100

# 종목별 팩터 프로파일 생성 (실제: yfinance로 수익률 계산 후 팩터 산출)
# 3개의 자연 군집: 성장주, 가치주, 배당주
growth_stocks  = np.random.multivariate_normal([1.5, -0.5, 0.2, -1.0], np.eye(4)*0.3, 35)
value_stocks   = np.random.multivariate_normal([-1.0, 1.5, -0.3, 0.5], np.eye(4)*0.3, 35)
dividend_stocks= np.random.multivariate_normal([-0.5, 0.8, 1.5, 0.8], np.eye(4)*0.3, 30)

X_stocks = np.vstack([growth_stocks, value_stocks, dividend_stocks])
true_labels = np.array([0]*35 + [1]*35 + [2]*30)
factor_names = ['모멘텀', 'PBR역수', '배당수익률', '변동성역수']

# 종목 이름 (예시)
stock_names = [f"종목{i+1:03d}" for i in range(n_stocks)]

scaler = StandardScaler()
X_scaled = scaler.fit_transform(X_stocks)
```

### 최적 K 선택: 엘보우 + 실루엣

```python
k_range = range(2, 11)
inertias, silhouettes, ch_scores = [], [], []

for k in k_range:
    km = KMeans(n_clusters=k, n_init=20, random_state=42)
    labels = km.fit_predict(X_scaled)
    inertias.append(km.inertia_)
    silhouettes.append(silhouette_score(X_scaled, labels))
    ch_scores.append(calinski_harabasz_score(X_scaled, labels))

fig, axes = plt.subplots(1, 3, figsize=(15, 4))

axes[0].plot(k_range, inertias, 'bo-')
axes[0].set_xlabel('K'); axes[0].set_ylabel('Inertia')
axes[0].set_title('엘보우 방법 (꺾이는 지점이 최적 K)')
axes[0].axvline(x=3, color='r', linestyle='--', label='K=3')
axes[0].legend()

axes[1].plot(k_range, silhouettes, 'go-')
axes[1].set_xlabel('K'); axes[1].set_ylabel('Silhouette Score')
axes[1].set_title('실루엣 점수 (높을수록 좋음)')

axes[2].plot(k_range, ch_scores, 'ro-')
axes[2].set_xlabel('K'); axes[2].set_ylabel('CH Score')
axes[2].set_title('Calinski-Harabasz Score')

plt.tight_layout()
plt.savefig('kmeans_k_selection.png', dpi=120)
print("저장: kmeans_k_selection.png")
print(f"\n최적 실루엣 점수: K={list(k_range)[np.argmax(silhouettes)]}, Score={max(silhouettes):.4f}")
```

### K-Means 결과 해석

```python
best_k = 3
km_final = KMeans(n_clusters=best_k, n_init=20, random_state=42)
cluster_labels = km_final.fit_predict(X_scaled)

# PCA로 2D 시각화
pca = PCA(n_components=2)
X_pca = pca.fit_transform(X_scaled)

colors = ['#E74C3C', '#2ECC71', '#3498DB']
cluster_names = ['성장주', '가치주', '배당주']

plt.figure(figsize=(10, 6))
for c in range(best_k):
    mask = cluster_labels == c
    plt.scatter(X_pca[mask, 0], X_pca[mask, 1],
                c=colors[c], label=f'군집 {c} ({cluster_names[c]})',
                alpha=0.7, s=60)

# 중심점 표시
centers_pca = pca.transform(km_final.cluster_centers_)
plt.scatter(centers_pca[:, 0], centers_pca[:, 1],
            c='black', marker='★', s=200, zorder=5, label='군집 중심')

plt.xlabel(f'PC1 ({pca.explained_variance_ratio_[0]:.1%})')
plt.ylabel(f'PC2 ({pca.explained_variance_ratio_[1]:.1%})')
plt.title('종목 군집화 결과 (PCA 2D 시각화)')
plt.legend()
plt.tight_layout()
plt.savefig('kmeans_pca.png', dpi=120)
print("저장: kmeans_pca.png")

# 군집별 팩터 프로파일 해석
df_clusters = pd.DataFrame(X_stocks, columns=factor_names)
df_clusters['군집'] = cluster_labels
print("\n군집별 평균 팩터 프로파일:")
print(df_clusters.groupby('군집')[factor_names].mean().round(3))
```

## 3. DBSCAN: 밀도 기반 클러스터링

K-Means와 달리 K를 사전에 정할 필요 없고, 임의 형태의 군집과 이상치를 자동 탐지.

```
ε (epsilon): 이웃으로 간주하는 최대 거리
min_samples: 핵심 포인트가 되기 위한 최소 이웃 수

Core Point: ε 반경 내 min_samples 이상의 이웃 보유
Border Point: Core Point의 ε 이웃이지만 자신은 Core가 아닌 점
Noise Point: 어떤 Core Point에도 속하지 않는 이상치 → 레이블 = -1
```

```python
from sklearn.neighbors import NearestNeighbors

# ε 선택: k-거리 그래프 (k = min_samples)
min_samples = 5
nbrs = NearestNeighbors(n_neighbors=min_samples).fit(X_scaled)
distances, _ = nbrs.kneighbors(X_scaled)
k_distances = np.sort(distances[:, -1])[::-1]

plt.figure(figsize=(8, 4))
plt.plot(k_distances)
plt.axhline(y=1.0, color='r', linestyle='--', label='ε=1.0 (꺾이는 지점)')
plt.xlabel('포인트 (정렬됨)')
plt.ylabel(f'{min_samples}-번째 이웃 거리')
plt.title('k-거리 그래프로 최적 ε 선택')
plt.legend()
plt.tight_layout()
plt.savefig('dbscan_epsilon.png', dpi=120)

# DBSCAN 적용
dbscan = DBSCAN(eps=1.0, min_samples=5)
db_labels = dbscan.fit_predict(X_scaled)

n_clusters = len(set(db_labels)) - (1 if -1 in db_labels else 0)
n_noise    = (db_labels == -1).sum()
print(f"\nDBSCAN 결과: {n_clusters}개 군집, {n_noise}개 이상치")
print(f"군집 크기: {pd.Series(db_labels[db_labels != -1]).value_counts().to_dict()}")
```

### 이상 거래 탐지 활용

## 4. 계층적 클러스터링 (덴드로그램)

```python
from scipy.cluster.hierarchy import dendrogram, linkage
from sklearn.cluster import AgglomerativeClustering

# 덴드로그램으로 자연 군집 수 시각화
linkage_matrix = linkage(X_scaled, method='ward')

plt.figure(figsize=(12, 5))
dendrogram(linkage_matrix,
           labels=[f"S{i}" for i in range(n_stocks)],
           color_threshold=5.0,
           truncate_mode='lastp', p=30,  # 상위 30개 노드만 표시
           leaf_rotation=90, leaf_font_size=8)
plt.title('Ward 연결법 덴드로그램')
plt.xlabel('종목')
plt.ylabel('거리')
plt.axhline(y=5.0, color='r', linestyle='--', label='군집 자르기 임계값')
plt.legend()
plt.tight_layout()
plt.savefig('dendrogram.png', dpi=120)
print("저장: dendrogram.png")

# 계층적 클러스터링 (K=3)
agg = AgglomerativeClustering(n_clusters=3, linkage='ward')
agg_labels = agg.fit_predict(X_scaled)
print(f"\nAgglomerative Clustering 실루엣: {silhouette_score(X_scaled, agg_labels):.4f}")
```

## 5. 퀀트 실전: 코릴레이션 기반 종목 군집화

```python
# 실제 수익률 상관관계 기반 군집화
np.random.seed(0)
n_days, n_stocks = 252, 50

# 팩터 모델로 수익률 시뮬레이션 (공통 팩터 + 개별 노이즈)
market_factor = np.random.randn(n_days)
sector_factors = np.random.randn(n_days, 5)
sector_assignment = np.random.randint(0, 5, n_stocks)

returns_matrix = np.zeros((n_days, n_stocks))
for j in range(n_stocks):
    s = sector_assignment[j]
    returns_matrix[:, j] = (0.5 * market_factor +
                             0.3 * sector_factors[:, s] +
                             0.2 * np.random.randn(n_days))

# 상관계수 행렬 → 거리 행렬
corr_matrix = np.corrcoef(returns_matrix.T)
dist_matrix = np.sqrt(2 * (1 - corr_matrix))  # 상관계수 → 거리 변환

# 계층적 클러스터링으로 섹터 발견
from scipy.cluster.hierarchy import fcluster
linkage_ret = linkage(dist_matrix[np.triu_indices(n_stocks, k=1)], method='ward')

# TODO: 덴드로그램으로 자연 군집 확인 후 fcluster로 레이블 추출
labels_ret = fcluster(linkage_ret, t=5, criterion='distance')
print(f"발견된 섹터 수: {len(np.unique(labels_ret))}")
print(f"섹터별 종목 수: {pd.Series(labels_ret).value_counts().sort_index().to_dict()}")

# 포트폴리오 적용: 각 군집에서 1종목씩 선택 → 분산 효과 극대화
# (상관관계가 낮은 종목들로 포트폴리오 구성)
```

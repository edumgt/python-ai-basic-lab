## 1. 앙상블 방법론 전체 지도

```
앙상블 방법
├── Bagging (병렬)
│   ├── Random Forest
│   └── Extra Trees
├── Boosting (순차)
│   ├── AdaBoost
│   ├── Gradient Boosting (GBDT)
│   ├── XGBoost
│   ├── LightGBM ← 실무 표준
│   └── CatBoost
└── Stacking / Blending
```

### Bagging vs Boosting

| 구분 | Bagging | Boosting |
|------|---------|---------|
| 훈련 방식 | 병렬 (독립적) | 순차 (이전 오류 교정) |
| 목표 | 분산(Variance) 감소 | 편향(Bias) + 분산 감소 |
| 과적합 민감도 | 낮음 | 높음 (학습률·트리수 조절 필요) |
| 대표 모델 | Random Forest | XGBoost, LightGBM |

## 2. Gradient Boosting 원리

잔차(Residual)를 순차적으로 학습하며 예측 오류를 줄여나갑니다.

$$F_0(\mathbf{x}) = \text{초기 예측 (평균값)}$$
$$F_m(\mathbf{x}) = F_{m-1}(\mathbf{x}) + \eta \cdot h_m(\mathbf{x})$$

여기서 $h_m$은 $m$번째 트리로, **이전 잔차의 음의 그래디언트**를 학습합니다:
$$r_{im} = -\left[\frac{\partial L(y_i, F(\mathbf{x}_i))}{\partial F(\mathbf{x}_i)}\right]_{F=F_{m-1}}$$

- $\eta$ (학습률): 각 트리의 기여도. 작을수록 안정적, 더 많은 트리 필요.
- **핵심 통찰**: 트리 하나가 전체를 학습하는 것이 아니라, 많은 약한 학습기(weak learner)가 순차적으로 오류를 보완.

## 3. XGBoost 실습

XGBoost는 GBDT에 **정규화 항**, **병렬 트리 구축**, **결측치 자동 처리**를 추가.

$$\mathcal{L}^{(t)} = \sum_i l(y_i, \hat{y}_i^{(t-1)} + f_t(\mathbf{x}_i)) + \Omega(f_t) \quad \text{where } \Omega(f) = \gamma T + \frac{1}{2}\lambda\|\mathbf{w}\|^2$$

```python
import numpy as np
import pandas as pd
from xgboost import XGBClassifier
from sklearn.metrics import roc_auc_score
from sklearn.model_selection import TimeSeriesSplit
import matplotlib.pyplot as plt

np.random.seed(42)
n = 1000

# 금융 팩터 시뮬레이션 (12개 팩터)
factor_names = [
    'mom_5', 'mom_20', 'mom_60',       # 모멘텀
    'vol_10', 'vol_20',                 # 변동성
    'rsi_14', 'macd_signal',            # 기술적 지표
    'size', 'value', 'quality',         # 팩터 스타일
    'turnover', 'short_interest'        # 거래 특성
]
X = np.random.randn(n, len(factor_names))
y = (0.3 * X[:, 0] + 0.2 * X[:, 2] - 0.15 * X[:, 3] +
     0.1 * X[:, 7] + np.random.randn(n) * 0.5 > 0).astype(int)

split = int(n * 0.8)
X_tr, X_te = X[:split], X[split:]
y_tr, y_te = y[:split], y[split:]

# XGBoost 기본 모델
xgb = XGBClassifier(
    n_estimators=300,
    learning_rate=0.05,      # 낮은 학습률 + 많은 트리 = 안정적
    max_depth=4,             # 얕은 트리 (과적합 방지)
    min_child_weight=20,     # 금융 데이터의 노이즈 대응
    subsample=0.8,           # 행 서브샘플링
    colsample_bytree=0.8,    # 열 서브샘플링 (특성 다양성)
    reg_alpha=0.1,           # L1 정규화
    reg_lambda=1.0,          # L2 정규화
    eval_metric='auc',
    early_stopping_rounds=30,
    random_state=42,
    n_jobs=-1,
)

xgb.fit(
    X_tr, y_tr,
    eval_set=[(X_te, y_te)],
    verbose=50,
)

y_prob = xgb.predict_proba(X_te)[:, 1]
print(f"\nXGBoost 테스트 AUC: {roc_auc_score(y_te, y_prob):.4f}")
print(f"최적 트리 수: {xgb.best_iteration}")
```

## 4. LightGBM — 실무 표준

Random Forest와 XGBoost 대비 **대용량 데이터에서 빠르고 메모리 효율적**.

### XGBoost vs LightGBM 핵심 차이

| 구분 | XGBoost | LightGBM |
|------|---------|---------|
| 트리 성장 방식 | Level-wise (깊이 우선) | Leaf-wise (손실 우선) |
| 학습 속도 | 보통 | 매우 빠름 |
| 메모리 | 많이 사용 | 효율적 (히스토그램 기반) |
| 과적합 | 안정적 | `min_child_samples` 주의 |
| 범주형 특성 | 인코딩 필요 | 네이티브 지원 |

```python
from lightgbm import LGBMClassifier, early_stopping, log_evaluation

lgb = LGBMClassifier(
    n_estimators=500,
    learning_rate=0.03,
    num_leaves=31,           # 최대 리프 수 (2^max_depth 참고값)
    max_depth=-1,            # 제한 없음 (num_leaves로 제어)
    min_child_samples=30,    # 리프 최소 샘플 수 (과적합 방지 핵심)
    subsample=0.8,
    colsample_bytree=0.8,
    reg_alpha=0.1,
    reg_lambda=1.0,
    random_state=42,
    n_jobs=-1,
    verbose=-1,
)

lgb.fit(
    X_tr, y_tr,
    eval_set=[(X_te, y_te)],
    eval_metric='auc',
    callbacks=[
        early_stopping(stopping_rounds=30, verbose=False),
        log_evaluation(period=100),
    ],
)

y_prob_lgb = lgb.predict_proba(X_te)[:, 1]
print(f"LightGBM 테스트 AUC: {roc_auc_score(y_te, y_prob_lgb):.4f}")
print(f"최적 트리 수: {lgb.best_iteration_}")
```

## 5. Optuna를 활용한 하이퍼파라미터 최적화

GridSearchCV보다 효율적인 **베이즈 최적화** 기반 탐색.

```python
# pip install optuna
import optuna
optuna.logging.set_verbosity(optuna.logging.WARNING)

tscv = TimeSeriesSplit(n_splits=5)

def objective(trial):
    params = {
        'n_estimators':      trial.suggest_int('n_estimators', 100, 500),
        'learning_rate':     trial.suggest_float('learning_rate', 0.01, 0.3, log=True),
        'num_leaves':        trial.suggest_int('num_leaves', 15, 63),
        'min_child_samples': trial.suggest_int('min_child_samples', 10, 100),
        'subsample':         trial.suggest_float('subsample', 0.6, 1.0),
        'colsample_bytree':  trial.suggest_float('colsample_bytree', 0.6, 1.0),
        'reg_alpha':         trial.suggest_float('reg_alpha', 1e-3, 10.0, log=True),
        'reg_lambda':        trial.suggest_float('reg_lambda', 1e-3, 10.0, log=True),
        'random_state': 42, 'n_jobs': -1, 'verbose': -1,
    }
    model = LGBMClassifier(**params)

    scores = []
    for tr_idx, val_idx in tscv.split(X_tr):
        model.fit(X_tr[tr_idx], y_tr[tr_idx])
        prob = model.predict_proba(X_tr[val_idx])[:, 1]
        scores.append(roc_auc_score(y_tr[val_idx], prob))
    return np.mean(scores)

study = optuna.create_study(direction='maximize',
                             sampler=optuna.samplers.TPESampler(seed=42))
study.optimize(objective, n_trials=50, show_progress_bar=True)

print(f"\n최적 AUC: {study.best_value:.4f}")
print(f"최적 파라미터:\n{study.best_params}")

# 최적 파라미터로 최종 모델 훈련
best_lgb = LGBMClassifier(**study.best_params, random_state=42)
best_lgb.fit(X_tr, y_tr)
final_auc = roc_auc_score(y_te, best_lgb.predict_proba(X_te)[:, 1])
print(f"최종 테스트 AUC: {final_auc:.4f}")
```

## 6. 모델 스태킹 (Stacking)

```python
from sklearn.ensemble import StackingClassifier
from sklearn.linear_model import LogisticRegression
from sklearn.svm import SVC
from sklearn.ensemble import RandomForestClassifier

base_models = [
    ('rf',   RandomForestClassifier(n_estimators=100, max_depth=4, random_state=42)),
    ('xgb',  XGBClassifier(n_estimators=100, max_depth=3, random_state=42, verbosity=0)),
    ('lgb',  LGBMClassifier(n_estimators=100, max_depth=3, random_state=42, verbose=-1)),
]
meta_model = LogisticRegression(C=1.0)

stacker = StackingClassifier(
    estimators=base_models,
    final_estimator=meta_model,
    cv=TimeSeriesSplit(n_splits=5),  # 시계열 교차 검증 사용
    passthrough=False,
    n_jobs=-1,
)
stacker.fit(X_tr, y_tr)
stack_auc = roc_auc_score(y_te, stacker.predict_proba(X_te)[:, 1])
print(f"Stacking AUC: {stack_auc:.4f}")
```

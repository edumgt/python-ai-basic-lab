## 1. 신경망 기본 구조

```
입력층          은닉층 1         은닉층 2        출력층
  x₁ ──────→ [뉴런 × 128] ──→ [뉴런 × 64] ──→ ŷ (수익률)
  x₂ ────↗↘          ↗↘              ↗↘
  ...
  xₚ ──────→                               ──→
```

$$\text{Layer}_l: \mathbf{a}^{(l)} = \sigma\left(\mathbf{W}^{(l)}\mathbf{a}^{(l-1)} + \mathbf{b}^{(l)}\right)$$

### 활성화 함수 비교

| 함수 | 수식 | 특징 | 사용 위치 |
|------|------|------|---------|
| ReLU | $\max(0, x)$ | 기울기 소실 방지, 계산 빠름 | **은닉층 기본값** |
| LeakyReLU | $\max(0.01x, x)$ | Dead ReLU 방지 | 깊은 네트워크 |
| Sigmoid | $\frac{1}{1+e^{-x}}$ | [0,1] 출력 | 이진 분류 출력층 |
| Tanh | $\tanh(x)$ | [-1,1] 출력 | 시계열 (LSTM) |
| Softmax | $\frac{e^{x_i}}{\sum e^{x_j}}$ | 확률 합=1 | 다중 분류 출력층 |

---

## 2. 역전파 (Backpropagation)

$$\frac{\partial L}{\partial \mathbf{W}^{(l)}} = \frac{\partial L}{\partial \mathbf{a}^{(l)}} \cdot \frac{\partial \mathbf{a}^{(l)}}{\partial \mathbf{z}^{(l)}} \cdot \frac{\partial \mathbf{z}^{(l)}}{\partial \mathbf{W}^{(l)}}$$

**Adam Optimizer**: 각 파라미터마다 적응적 학습률 적용. 금융 데이터처럼 비정상(non-stationary) 시계열에서 SGD보다 안정적.

$$m_t = \beta_1 m_{t-1} + (1-\beta_1)\nabla_t, \quad v_t = \beta_2 v_{t-1} + (1-\beta_2)\nabla_t^2$$
$$\theta_t = \theta_{t-1} - \frac{\eta}{\sqrt{\hat{v}_t}+\epsilon}\hat{m}_t$$

## 3. PyTorch로 MLP 구현

### 데이터 준비

```python
import numpy as np
import torch
import torch.nn as nn
from torch.utils.data import DataLoader, TensorDataset
from sklearn.preprocessing import StandardScaler
from sklearn.metrics import roc_auc_score
import matplotlib.pyplot as plt

np.random.seed(42)
torch.manual_seed(42)

n, n_features = 2000, 15
X_raw = np.random.randn(n, n_features)
# 비선형 관계 포함 (MLP가 선형 모델보다 유리한 상황)
y = (0.3 * X_raw[:, 0]**2 +
     0.4 * X_raw[:, 1] * X_raw[:, 2] +
     0.2 * np.sin(X_raw[:, 3]) +
     np.random.randn(n) * 0.5 > 0).astype(np.float32)

scaler = StandardScaler()
X_scaled = scaler.fit_transform(X_raw).astype(np.float32)

split_tr = int(n * 0.7)
split_val = int(n * 0.85)

X_tr  = torch.tensor(X_scaled[:split_tr])
X_val = torch.tensor(X_scaled[split_tr:split_val])
X_te  = torch.tensor(X_scaled[split_val:])
y_tr  = torch.tensor(y[:split_tr])
y_val = torch.tensor(y[split_tr:split_val])
y_te  = torch.tensor(y[split_val:])

train_loader = DataLoader(TensorDataset(X_tr, y_tr), batch_size=64, shuffle=True)
val_loader   = DataLoader(TensorDataset(X_val, y_val), batch_size=256)
```

### MLP 모델 정의

```python
class QuantMLP(nn.Module):
    def __init__(self, input_dim: int, hidden_dims: list, dropout: float = 0.3):
        super().__init__()
        layers = []
        prev_dim = input_dim
        for h_dim in hidden_dims:
            layers += [
                nn.Linear(prev_dim, h_dim),
                nn.BatchNorm1d(h_dim),  # 배치 정규화: 학습 안정화
                nn.ReLU(),
                nn.Dropout(dropout),   # 과적합 방지
            ]
            prev_dim = h_dim
        layers.append(nn.Linear(prev_dim, 1))  # 출력층 (sigmoid 없음 → BCEWithLogitsLoss 사용)
        self.net = nn.Sequential(*layers)

    def forward(self, x):
        return self.net(x).squeeze(-1)

model = QuantMLP(input_dim=n_features, hidden_dims=[128, 64, 32], dropout=0.3)
print(model)
print(f"파라미터 수: {sum(p.numel() for p in model.parameters()):,}")
```

### 훈련 루프

```python
criterion = nn.BCEWithLogitsLoss()
optimizer = torch.optim.Adam(model.parameters(), lr=1e-3, weight_decay=1e-4)
scheduler = torch.optim.lr_scheduler.ReduceLROnPlateau(optimizer, patience=10, factor=0.5)

def run_epoch(loader, train=True):
    if train:
        model.train()
    else:
        model.eval()

    total_loss, preds, targets = 0, [], []
    with torch.set_grad_enabled(train):
        for X_b, y_b in loader:
            logits = model(X_b)
            loss = criterion(logits, y_b)
            if train:
                optimizer.zero_grad()
                loss.backward()
                nn.utils.clip_grad_norm_(model.parameters(), max_norm=1.0)
                optimizer.step()
            total_loss += loss.item() * len(X_b)
            preds.extend(torch.sigmoid(logits).detach().numpy())
            targets.extend(y_b.numpy())

    avg_loss = total_loss / len(loader.dataset)
    auc = roc_auc_score(targets, preds)
    return avg_loss, auc

n_epochs = 100
train_history = {'loss': [], 'auc': []}
val_history   = {'loss': [], 'auc': []}
best_val_auc  = 0
best_weights  = None

for epoch in range(1, n_epochs + 1):
    tr_loss, tr_auc   = run_epoch(train_loader, train=True)
    val_loss, val_auc = run_epoch(val_loader, train=False)
    scheduler.step(val_loss)

    train_history['loss'].append(tr_loss)
    train_history['auc'].append(tr_auc)
    val_history['loss'].append(val_loss)
    val_history['auc'].append(val_auc)

    if val_auc > best_val_auc:
        best_val_auc = val_auc
        best_weights = {k: v.clone() for k, v in model.state_dict().items()}

    if epoch % 20 == 0:
        print(f"Epoch {epoch:3d} | TR Loss={tr_loss:.4f} AUC={tr_auc:.4f} | "
              f"VAL Loss={val_loss:.4f} AUC={val_auc:.4f}")

model.load_state_dict(best_weights)
```

## 5. 금융 데이터 MLP 실전 팁

### 레짐 인식 네트워크

```python
class RegimeAwareMLP(nn.Module):
    """시장 레짐(bull/bear/sideways)에 따라 다른 헤드 사용"""
    def __init__(self, input_dim: int):
        super().__init__()
        self.shared = nn.Sequential(
            nn.Linear(input_dim, 64), nn.ReLU(), nn.Dropout(0.2),
        )
        # 3개 레짐별 전문 헤드
        self.heads = nn.ModuleList([
            nn.Sequential(nn.Linear(64, 32), nn.ReLU(), nn.Linear(32, 1))
            for _ in range(3)
        ])
        # 레짐 분류기
        self.regime_classifier = nn.Linear(64, 3)

    def forward(self, x):
        shared_feat = self.shared(x)
        regime_logits = self.regime_classifier(shared_feat)
        regime_weights = torch.softmax(regime_logits, dim=-1)  # (B, 3)

        # 각 헤드의 예측을 레짐 확률로 가중 평균
        head_preds = torch.stack([h(shared_feat).squeeze(-1) for h in self.heads], dim=1)  # (B, 3)
        final_pred = (head_preds * regime_weights).sum(dim=1)
        return final_pred, regime_logits
```

## 핵심 요약

- **배치 정규화**: 학습 안정화, 학습률 민감도 감소
- **Dropout**: 훈련 시 랜덤하게 뉴런 비활성화 → 과적합 방지
- **Early Stopping**: Val Loss가 더 이상 개선되지 않으면 훈련 중단
- **Gradient Clipping**: `clip_grad_norm_` → 폭발하는 그래디언트 방지
- **BCEWithLogitsLoss**: 수치 안정성을 위해 Sigmoid + BCE를 합친 버전 사용

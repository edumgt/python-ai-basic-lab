# 웹앱 실습 7: 호텔 주가 멀티모델 실험실

> 계절성, 다수 특성, ML/DL 모델 비교를 카드형 대시보드에서 읽는 실습입니다.

---

## 이 실습의 목표

- 시계열형 특성, 계절성, 모델 비교를 한 화면에서 읽습니다.
- `호텔 예약률 + 주가`처럼 여러 변수가 섞인 문제를 어떻게 해석하는지 익힙니다.
- 계절성, 혼동행렬, 신호 같은 단어는 [voca.md](voca.md)와 함께 보면 더 쉽습니다.

## 🌟 초등학생도 이해할 수 있어요

이 실습은 단순히 주가만 보는 게 아니라

- 손님이 얼마나 오는지
- 계절이 여름인지 겨울인지
- 지난달보다 분위기가 어떤지

를 같이 보는 문제입니다.

쉽게 말해 **"호텔 숙제와 주가 숙제를 함께 푸는 AI"**를 보는 화면입니다.

## 이 화면의 장점

| 요소 | 장점 |
|------|------|
| 상단 지표 카드 | 성능 숫자를 빠르게 비교 |

| 탭 구조 | 중요도, 가격, 신호, 혼동행렬 분리 |

## 호텔-주가 실험실 단어 사전

| 단어 | 쉬운 뜻 | 이 화면에서 어디에 연결되나 |
|------|------|------|
| 계절성 | 계절이나 월에 따라 반복되는 흐름 | 여름/겨울, 성수기/비수기 차이를 볼 때 중요합니다. |
| 멀티특성 | 여러 입력 힌트를 같이 쓰는 것 | 예약률, 가격, 이벤트, 거래량을 함께 봅니다. |
| 신호(signal) | 모델이 지금 어떻게 행동하라고 주는 표시 | 매수/관망 같은 판단 흐름으로 읽을 수 있습니다. |
| 혼동 행렬 | 맞고 틀린 경우를 표로 나눈 것 | FP, FN이 어떤 식으로 생기는지 봅니다. |
| 시뮬레이션 | 실제처럼 흉내 내어 돌려 보는 것 | 과거 데이터에서 전략이 어땠을지 확인합니다. |
| MLP | 가장 기본적인 여러 층 신경망 | 이 화면의 DL 모델 비교에서 자주 등장합니다. |

---

## 실습 순서

1. 먼저 `랜덤 포레스트`로 실행합니다.
2. `특성 중요도` 탭에서 어떤 예약률/계절 특성이 중요한지 봅니다.
3. `주가 시뮬레이션` 탭에서 테스트 구간을 확인합니다.
4. `예측 신호` 탭에서 개별 시점 예측을 읽습니다.
5. `혼동 행렬` 탭에서 FP, FN이 어떤 식으로 나오는지 봅니다.
6. `신경망` 계열 모델로 바꿔 다시 실행합니다.
7. `신경망 시뮬레이션` 탭이 열리면 흐름을 한 번 재생합니다.

### 계절성이 왜 중요할까요?

호텔이나 여행 관련 데이터는

- 방학
- 휴가철
- 연말
- 행사 시즌

같이 반복되는 흐름을 많이 가집니다.

그래서 단순히 "어제보다 올랐나"만 보는 것보다  
"지금이 어떤 계절과 분위기인지"를 같이 보는 것이 더 자연스럽습니다.

## 관찰 질문

- 계절 특성이 중요하게 나오는가, 예약률 특성이 중요하게 나오는가?
- ML 모델과 DL 모델 중 어떤 쪽이 더 안정적으로 보이나요?
- `FP`와 `FN` 중 무엇이 더 많이 나오는지, 그것이 어떤 의미인지 설명할 수 있나요?

- ML 모델 하나와 DL 모델 하나를 골라 성능을 비교합니다.
- 중요 특성 상위 3개를 적고, 왜 그런 순서가 나왔을지 추측합니다.
- `혼동 행렬`을 보고 이 모델의 약점을 한 줄로 적습니다.

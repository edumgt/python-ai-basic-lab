# xor-train

- 원본 소스: `/home/runner/work/py-ml-dl-lab/py-ml-dl-lab/ch4/xor-train.py`
- 장: `CH4`

## 이 파일은 어떤 실습인가?
선형적으로 분리되지 않는 XOR 문제를 SVM으로 다뤄 보는 대표적인 입문 실습이다.

## 실습은 어떻게 하면 되나?
1. `python /home/runner/work/py-ml-dl-lab/py-ml-dl-lab/ch4/xor-train.py` 를 실행한다.
2. 학습 후 4개 입력에 대한 예측을 확인한다.

## 사용되는 ML / DL / 데이터 처리 알고리즘
`sklearn.svm.SVC`를 사용하는 이진 분류 실습이다. XOR처럼 단순한 예제로 비선형 분류기의 필요성을 설명한다.

## 입력과 출력
입력은 코드 안에 직접 정의된 4개의 XOR 샘플, 출력은 예측 결과와 정확도다.

## 실습 포인트
머신러닝 분류기의 가장 작은 예제로 이해하면 된다.

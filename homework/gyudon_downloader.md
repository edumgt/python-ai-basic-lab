# gyudon_downloader


## 이 파일은 어떤 실습인가?
규동 이미지 검색 및 다운로드를 자동화해 데이터셋을 수집하는 실습이다.

## 실습은 어떻게 하면 되나?
1. `python /home/runner/work/py-ml-dl-lab/py-ml-dl-lab/ch7/gyudon_downloader.py` 를 실행한다.
2. 생성된 이미지 폴더를 확인한 뒤 데이터셋 생성 스크립트로 이어간다.

## 사용되는 ML / DL / 데이터 처리 알고리즘
ML 알고리즘은 없고 웹 API 검색/다운로드 자동화만 수행한다.

## 입력과 출력
입력은 검색어와 원격 API 응답, 출력은 `image/gyudon/` 이미지 파일들이다.

## 실습 포인트
학습 데이터 수집 단계까지 실습 범위를 확장한 예제다.

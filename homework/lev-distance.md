# lev-distance


## 이 파일은 어떤 실습인가?
편집 거리(Levenshtein distance)를 직접 구현해 문자열 유사도를 계산하는 실습이다.

## 실습은 어떻게 하면 되나?
1. `python /home/runner/work/py-ml-dl-lab/py-ml-dl-lab/ch6/lev-distance.py` 를 실행한다.

## 사용되는 ML / DL / 데이터 처리 알고리즘
동적 계획법 기반 Levenshtein distance를 사용한다.

## 입력과 출력
입력은 코드 안의 문자열 목록, 출력은 거리 계산 결과와 정렬 결과다.

## 실습 포인트
오탈자 교정, 문자열 매칭 기초 개념을 배우는 예제다.

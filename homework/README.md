## 각 py 파일은 개인별 환경, 필요한 파일 경로 오류, 파일 자체의 오탈자 등으로 오류가 있는 파일이며, 해당 수정 후 결과를 캡쳐하여 본인 repo 에 저장한다.
## 수정 파일의 내용을 주식 관련 업무로 변경한다.

- `ch01` 기초 연산/텐서 감각
- `ch02` 데이터프레임·전처리 기초
- `ch03` 지도학습 입문
- `ch04` 검증/분류 확장
- `ch05` 데이터 준비·허브형 기본 실습
- `ch06` 신경망 기초 실습
- `ch07` 텍스트/NLP 기초
- `ch08` 시퀀스·딥러닝 NLP
- `ch09` 챗봇/웹 연동 스크립트
- `ch10` OCR 실습
- `ch11` 이미지 분류 실습
- `ch12` 비전 응용/유사도 실습

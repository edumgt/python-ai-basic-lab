# botengine


## 이 파일은 어떤 실습인가?
간단한 규칙/마르코프 기반 챗봇 엔진을 만드는 실습이다.

## 실습은 어떻게 하면 되나?
1. 일반적으로 `chatbot.py`에서 호출해 사용한다.
2. 필요한 사전 파일이 준비된 상태에서 `make_reply()`를 호출해 응답을 생성한다.

## 사용되는 ML / DL / 데이터 처리 알고리즘
형태소 분석으로 토큰을 추출한 뒤 3-gram 수준의 딕셔너리를 만들고, 무작위 선택을 통해 응답 문장을 생성하는 Markov chain 계열 기법이다.

## 입력과 출력
입력은 사용자 문장과 사전/사전 캐시 파일, 출력은 생성된 답변 문자열이다.

## 실습 포인트
딥러닝 챗봇이 아니라 통계적 문장 생성 기반 챗봇 예제다.

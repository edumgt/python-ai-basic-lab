# np-test-calc


## 이 파일은 어떤 실습인가?
NumPy 배열 연산과 평균 계산을 익히는 기초 실습이다.

## 실습은 어떻게 하면 되나?
1. `python /home/runner/work/py-ml-dl-lab/py-ml-dl-lab/ch5/np-test-calc.py` 를 실행한다.
2. 출력되는 배열과 평균값을 확인한다.

## 사용되는 ML / DL / 데이터 처리 알고리즘
ML 알고리즘은 없고 NumPy 수치 연산만 사용한다.

## 입력과 출력
입력은 코드 내부 배열, 출력은 계산 결과다.

## 실습 포인트
머신러닝 라이브러리들이 사용하는 기본 수치 계산 환경을 익히는 예제다.

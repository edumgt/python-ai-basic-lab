# chatbot

## 이 파일은 어떤 실습인가?
챗봇 엔진을 웹 폼에 연결해 질의응답 형태로 체험하는 실습이다.

## 실습은 어떻게 하면 되나?
1. CGI를 지원하는 웹 서버 환경에 배치한다.
2. 브라우저에서 질문을 입력하면 `botengine.make_reply()` 결과를 확인할 수 있다.

## 사용되는 ML / DL / 데이터 처리 알고리즘
알고리즘은 내부적으로 `botengine.py`의 마르코프/형태소 기반 응답 생성에 의존한다.

## 입력과 출력
입력은 웹 폼 질문 문자열, 출력은 웹 페이지에 렌더링된 응답이다.

## 실습 포인트
NLP 엔진과 사용자 인터페이스를 연결하는 배포형 실습이다.

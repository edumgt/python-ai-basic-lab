# lang-Webapp

- 원본 소스: `/home/runner/work/py-ml-dl-lab/py-ml-dl-lab/ch4/lang-Webapp.py`
- 장: `CH4`

## 이 파일은 어떤 실습인가?
학습된 언어 판별 모델을 웹 애플리케이션 형태로 사용해 보는 배포형 실습이다.

## 실습은 어떻게 하면 되나?
1. `lang-train.py`와 `lang-train-save.py`를 실행해 `freq.pkl` 모델 파일을 준비한다.
2. CGI를 지원하는 웹 서버 환경에 스크립트를 배치한다.
3. 브라우저에서 문장을 입력하고 어떤 언어로 분류되는지 확인한다.

## 사용되는 ML / DL / 데이터 처리 알고리즘
알고리즘 자체는 저장된 SVM 언어 분류 모델이며, 입력 문장을 알파벳 출현 빈도 벡터로 변환해 예측한다.

## 입력과 출력
입력은 웹 폼에 입력한 문자열과 `freq.pkl` 모델, 출력은 웹 페이지에 표시되는 예측 언어 결과다.

## 실습 포인트
학습 모델을 실제 인터페이스에 연결하는 흐름을 익히는 데 목적이 있다.

# bayes


## 이 파일은 어떤 실습인가?
한국어 텍스트를 나이브 베이즈 방식으로 분류하는 알고리즘을 직접 구현해 보는 실습이다.

## 실습은 어떻게 하면 되나?
1. 보통 `bayes_test.py`에서 import해서 사용한다.
2. 직접 사용할 경우 `BayesianFilter` 인스턴스를 만들고 `fit`, `predict`를 호출한다.
3. 간단한 스팸/중요 메일 같은 텍스트 분류 예제로 실험한다.

## 사용되는 ML / DL / 데이터 처리 알고리즘
KoNLPy 형태소 분석 결과를 기반으로 단어 빈도를 세고, 라플라스 스무딩을 적용한 Multinomial Naive Bayes를 로그 확률로 계산한다.

## 입력과 출력
입력은 한국어 문장과 카테고리 라벨, 출력은 예측 카테고리와 점수다.

## 실습 포인트
라이브러리 없이 분류기 원리를 직접 구현해 보는 교육용 코드다.

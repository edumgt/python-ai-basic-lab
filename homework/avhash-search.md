## 이 파일은 어떤 실습인가?
평균 해시와 해밍 거리로 비슷한 이미지를 검색하는 실습이다.

## 실습은 어떻게 하면 되나?
1. 검색 대상 이미지 폴더를 준비한다.
2. `python /home/runner/work/py-ml-dl-lab/py-ml-dl-lab/ch7/avhash-search.py` 를 실행한다.
3. 생성된 HTML 결과에서 유사 이미지 순위를 확인한다.

## 사용되는 ML / DL / 데이터 처리 알고리즘
Average Hash로 각 이미지를 표현하고 해밍 거리로 유사도를 계산한다.

## 입력과 출력
입력은 이미지 폴더와 질의 이미지, 출력은 유사도 순위 및 HTML 리포트다.

## 실습 포인트
콘텐츠 기반 이미지 검색(CBIR)의 아주 단순한 형태를 보여준다.

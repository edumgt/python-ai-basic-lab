# tb-bmi2


## 이 파일은 어떤 실습인가?
BMI 학습 그래프를 `name_scope`로 구조화해 TensorBoard에서 더 읽기 쉽게 만드는 실습이다.

## 실습은 어떻게 하면 되나?
1. `python /home/runner/work/py-ml-dl-lab/py-ml-dl-lab/ch5/tb-bmi2.py` 를 실행한다.
2. TensorBoard에서 interface, loss, training, accuracy 등의 그룹을 확인한다.

## 사용되는 ML / DL / 데이터 처리 알고리즘
알고리즘은 동일한 소프트맥스 회귀이며, 그래프 시각화 구조화가 핵심 차이점이다.

## 입력과 출력
입력은 `ch5/bmi.csv`, 출력은 정확도와 구조화된 TensorBoard 로그다.

## 실습 포인트
복잡한 그래프를 관리하는 방법을 학습할 수 있다.

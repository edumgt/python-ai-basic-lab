# facedetect


## 이 파일은 어떤 실습인가?
이미지에서 얼굴 위치를 검출하는 전통적 컴퓨터 비전 실습이다.

## 실습은 어떻게 하면 되나?
1. `python /home/runner/work/py-ml-dl-lab/py-ml-dl-lab/ch7/facedetect.py` 를 실행한다.

## 사용되는 ML / DL / 데이터 처리 알고리즘
OpenCV Haar Cascade 분류기를 이용한 얼굴 검출이다.

## 입력과 출력
입력은 사진 파일, 출력은 사각형이 그려진 결과 이미지다.

## 실습 포인트
딥러닝 이전 얼굴 검출 기법의 대표 예제다.
